# 課程總覽：20 小時完整學習路線
> 從零開始到獨立製作——循序漸進，每個模組都有可交付的成果

## 課程地圖

[flow]
1. 模組 1（2 小時）— 入門準備：軟體安裝、介面認識、工作環境設定
2. 模組 2（2 小時）— 媒體管理：專案設定、素材整理、代理工作流程
3. 模組 3（4 小時）— 剪輯基礎：Cut 快剪、Edit 精剪、轉場與字幕
4. 模組 4（3 小時）— 進階剪輯：多機剪輯、速度技巧、效果應用
5. 模組 5（4 小時）— 調色技術：色彩理論、一二級調色、LUT 應用
6. 模組 6（2.5 小時）— 音頻後製：人聲處理、音效設計、混音技術
7. 模組 7（1.5 小時）— Fusion 特效：節點系統、動態字幕製作
8. 模組 8（1 小時）— 輸出交付：格式選擇、平台設定、最終成果
[/flow]

### 學習成果

完成本課程後，你將能夠：

- 獨立完成一支 3-5 分鐘影片的全流程製作
- 掌握調色節點系統，打造電影質感
- 處理人聲降噪、混音，讓音質達到專業水準
- 輸出符合 YouTube、Instagram、廣播等各平台規格的影片

## 1-1 認識 DaVinci Resolve（30 分鐘）

### DaVinci Resolve 的定位

[tags]
- [blue] 起源：Blackmagic Design 開發，最初只有調色功能
- [orange] 現況：整合剪輯、調色、特效、音頻的全功能後製平台
- [green] 使用者：好萊塢電影、Netflix 劇集、廣告、婚攝、Vlogger 均在使用

> DaVinci Resolve 免費、跨平台，且調色功能是三者中最強的。對剛入門的學習者，免費版完全夠用；未來有需要再升級 Studio 版。

# 模組 2：媒體管理與專案設定
> 良好的素材管理習慣，是專業工作者與業餘者之間最大的差距

## 2-1 建立專案與時間軸設定（45 分鐘）

### 專案管理架構

```prompt [label="建議的專案資料夾結構"]
專案名稱/
├── 01_素材/
│   ├── A-Roll/     (主畫面：訪談、說話)
│   ├── B-Roll/     (補充畫面：環境、特寫)
│   ├── 音樂/
│   └── 音效/
├── 02_DaVinci 專案/   (存放 .dra 專案檔)
├── 03_輸出/           (最終渲染結果)
└── 04_備份/
```

### 時間軸格式設定

[tags]
- [blue] 解析度：1920×1080（Full HD）/ 3840×2160（4K）
- [orange] 幀率：24fps（電影感）/ 30fps（日常影片）/ 60fps（運動、遊戲）
- [green] 色彩空間：Rec.709（一般攝影機）/ DCI-P3（電影院標準）
- [purple] 音頻：48kHz / 24-bit（廣播電視標準）
[/tags]

[callout type="warning" title="幀率不可中途更改"]
幀率一旦設定不要中途更改。如果時間軸幀率與素材幀率不同，DaVinci 會做轉換，但可能造成動作不自然或影格重複。

拍攝前就確認好目標幀率，剪輯開始時就設定正確，可以避免很多後期問題。
[/callout]

## 2-2 媒體庫整理技巧（45 分鐘）

### 素材標記系統

[flow]
1. 顏色標記 — 紅：最佳鏡頭 / 黃：備選 / 灰：廢棄
2. 旗標系統 — 旗標素材為「已審閱」，無旗標為「待確認」
3. 關鍵字（Keyword）— 為素材加上「人物名稱」「地點」「動作」等關鍵字
4. 智慧媒體夾 — 根據條件自動聚集素材（例如：所有已旗標的 4K 素材）
5. 入出點標記 — 在素材上標記 In/Out 點，快速定位最佳片段
[/flow]

### 素材整理效率技巧

- 導入素材後立即做「媒體整理」，不要等到剪到一半才整理
- 為攝影機素材重新命名：`CAM_A_001_地點_場景` 格式
- 音頻素材單獨放在「音效」資料夾，與影片素材分開

### 專案工作流程進度

[steps-status]
- [done] 建立專案資料夾結構 | 按 01_素材、02_DaVinci 專案、03_輸出、04_備份 分類
- [done] 設定時間軸規格 | 解析度、幀率、色彩空間、音頻格式
- [done] 導入與整理素材 | 顏色標記、旗標、關鍵字分類
- [doing] 生成代理媒體 | 4K 素材轉 1/4 解析度，剪輯更流暢
- [todo] 開始剪輯 | 進入 Cut / Edit 頁面進行粗剪與精剪
[/steps-status]

## 2-3 代理媒體工作流程（30 分鐘）

### 什麼是代理媒體（Proxy）？

代理媒體是原始高解析度素材的低畫質版本，用於剪輯時降低電腦負擔：

[flow]
1. 導入 4K 原始素材 → DaVinci 自動生成低解析度代理檔（1080p 或更低）
2. 剪輯時使用代理檔 — 畫質低但播放流暢，所有剪輯操作都與原始素材同步
3. 調色與輸出時切換回原始素材 — 一鍵切換，所有剪輯點自動對應
4. 輸出高畫質成品
[/flow]

[callout type="info" title="為什麼要用代理媒體？"]
代理媒體不是「降低品質妥協」，而是「智慧工作流程」。好萊塢電影後製也使用類似的概念（稱為 Offline/Online Editing）。剪輯時使用低解析度檔案，最後輸出時換回原始素材，能大幅提升操作流暢度。
[/callout]

```prompt [label="啟用代理媒體設定"]
進入：專案設定 → 主設定 → 優化媒體與代理媒體
→ 代理媒體解析度：選「1/4 原始解析度」（4K 素材選此）
→ 代理媒體格式：DNxHR LB（Windows）或 ProRes 422 Proxy（macOS）
→ 在媒體資料庫選取所有素材 → 右鍵 → 生成代理媒體
```

# 模組 3：剪輯基礎
> 剪輯是說故事的藝術——工具操作只是手段，節奏感才是靈魂

## 3-1 Cut 頁面：快速粗剪（1 小時）

### Cut 頁面獨特設計

Cut 頁面是為快速交片而生，有兩個時間軸同時運作：

- **上方時間軸**：全片縮略圖，直接拖動快速定位
- **下方時間軸**：精確剪輯區，放大到幀級別操作
- **智慧插入（Smart Insert）**：自動找最近的剪切點，不用手動對齊

### Cut 頁面專屬工具

[tags]
- [blue] 智慧插入：拖素材到時間軸，自動對齊最近剪輯點
- [orange] 置前插入：強制插入到時間軸最前端
- [green] 附加（Append）：接在現有素材後方
- [purple] 合拍（Match Frame）：自動跳到與時間軸相同的素材位置
[/tags]

## 3-2 Edit 頁面：精剪技術（1.5 小時）

### Edit 頁面核心工具

[flow]
1. 選取工具（A）— 移動素材、調整長度
2. 修剪工具（T）— Ripple Edit：剪短後後續素材自動往前補位
3. 刀片工具（B）— 在任意位置切割素材
4. 滾動剪輯（R）— 同時縮短 A 素材末端、延長 B 素材開頭，總長不變
5. 滑移（Slip）— 素材位置不動，調整播放的起訖點
6. 滑動（Slide）— 播放範圍不動，調整素材在時間軸的位置
[/flow]

### 剪輯點的六種類型

```prompt [label="剪輯點類型說明"]
硬切（Hard Cut）   ：畫面直接切換，節奏最快，最常用
溶解（Dissolve）   ：畫面淡入淡出交疊，表達時間流逝
淡入淡出（Fade）   ：從黑幕淡入或淡出到黑幕，段落開始/結束
J-Cut             ：下一個鏡頭的音頻先出現，再切入畫面
L-Cut             ：上一個鏡頭的音頻延伸到下一個鏡頭畫面
Match Cut         ：利用相似構圖或動作連接兩個不同場景
```

[callout type="tip" title="剪輯的「三秒法則」"]
靜態鏡頭平均時長不超過三秒，動態鏡頭可適當延長。

超過三秒的靜態鏡頭，觀眾會開始注意到「這個鏡頭很長」，注意力從故事轉移到鏡頭本身——這是你不想要的。
[/callout]

### 三點剪輯（Three-Point Edit）

專業剪輯師必備技巧，只需指定三個點即可精確控制素材插入：

[flow]
1. 在來源監視器標記 In 點 — 素材要從哪裡開始使用
2. 在來源監視器標記 Out 點 — 素材要在哪裡結束
3. 在時間軸標記 In 點（或 Out 點）— 插入目標位置
4. 選擇插入方式（插入 / 覆蓋 / 替換）
[/flow]

[quiz type="single"]
Q: J-Cut 的特徵是什麼？
- [ ] 畫面先切換，聲音再跟著切換
- [x] 下一個鏡頭的聲音先出現，再切入畫面
- [ ] 畫面與聲音同時切換到下一個鏡頭
- [ ] 兩個畫面重疊溶解在一起
Hint: J-Cut 的「J」形狀來自時間軸上音頻提前於畫面的剪輯點。
[/quiz]

## 3-3 轉場效果與速度調整（1 小時）

### 轉場效果使用原則

[tags]
- [green] 適合使用：章節切換、時間跳躍、情緒轉換
- [orange] 避免過度使用：每個剪輯點都加轉場，會讓影片顯得廉價
- [blue] 最佳轉場長度：0.5-1 秒（太長轉場喧賓奪主）
- [purple] 最常用轉場：溶解（Dissolve）、Cross Fade、Dip to Black
[/tags]

### 速度調整技巧

```prompt [label="速度調整快捷操作"]
右鍵素材 → 「變速」→ 可選：
  固定速度：輸入百分比（200% = 2 倍速，50% = 慢動作）
  速度曲線：可在素材內做加減速，如從 50% 漸變到 200%
  
凍結幀（Freeze Frame）：
  右鍵剪輯點 → 「加入凍結幀」→ 可選凍結幀長度
  效果：畫面暫停，常用於強調、搞笑、情緒收尾
```

## 3-4 字幕與文字製作（30 分鐘）

### 字幕製作三種方法

[flow]
1. 手動字幕 — 效果庫 → 字幕 → 拖入時間軸，逐句輸入文字
2. 字幕軌（Subtitle Track）— 專用字幕軌，可批次匯入 SRT 字幕檔
3. AI 自動語音辨識 — Studio 版限定，自動生成字幕（免費版不支援）
[/flow]

```prompt [label="匯入 SRT 字幕檔步驟"]
1. 準備好 .srt 格式字幕文件（可用 Aegisub 軟體製作）
2. 時間軸上右鍵 → 「匯入字幕」→ 選取 .srt 檔案
3. DaVinci 自動建立字幕軌並填入所有文字
4. 可調整字幕字體、大小、位置、底色等樣式
```

# 模組 4：進階剪輯技術
> 掌握進階技巧，讓你的影片從「還不錯」升級到「令人印象深刻」

## 4-1 多機攝影剪輯（Multicam）（1 小時）

### 多機剪輯工作流程

多機剪輯適合訪談、演講、活動等有多台攝影機同時拍攝的場景：

[flow]
1. 同步素材 — 選取所有攝影機素材，右鍵 → 「建立多機剪輯」
2. 同步方式選擇 — 聲音波形（最準確）/ 時間碼 / 入點
3. 進入多機剪輯時間軸 — 上方顯示所有機位，下方為輸出時間軸
4. 實時剪輯 — 按播放，點選任一機位畫面即自動切換
5. 後期調整 — 可回頭修改任一剪輯點的機位選擇
[/flow]

[callout type="tip" title="打板（Clapper Board）的重要性"]
打板的聲音是最清晰的同步點。即使沒有正式打板，拍攝開始時拍一下手或做任何清晰的聲音動作，都能讓聲音同步更容易。

若現場條件不允許，DaVinci 的聲音波形自動同步功能已相當準確，但偶爾仍需手動微調。
[/callout]

## 4-2 速度曲線與創意剪輯技巧（1 小時）

### Ramp Speed（速度漸變）

速度漸變是影片中常見的電影感技巧：

```prompt [label="速度漸變設定步驟"]
1. 右鍵素材 → 「變速」→「自訂速度」
2. 在速度圖形（Retime Curve）視窗操作
3. 在需要改變速度的位置新增關鍵影格
4. 拖曳關鍵影格之間的曲線，創造加速/減速效果

常見應用：
  - 快速平移後急停：加速 → 凍結幀
  - 慢動作重點時刻：100% → 降至 20-30%
  - 節奏感切換：隨音樂拍點做速度變化
```

### 創意轉場技巧

[tags]
- [blue] 遮罩轉場（Wipe）：用物體遮住鏡頭時切換，製造無縫感
- [orange] 焦點轉場（Focus）：A 鏡頭拉失焦，B 鏡頭從失焦拉回清晰
- [green] 動作匹配（Match Cut）：利用相似的動作或形狀連接兩個場景
- [purple] 跳切（Jump Cut）：同一場景連續跳接，製造時間壓縮感
[/tags]

## 4-3 效果庫與插件應用（1 小時）

### 內建效果分類

[flow]
1. 色彩校正類 — Color Wheels、Curves、Hue vs Saturation
2. 模糊與銳化 — Gaussian Blur、Sharpen、Mosaic（馬賽克）
3. 風格化濾鏡 — Old Film、Vignette、Halation
4. 運動效果 — 穩定化、去搖晃（Stabilization）
5. ResolveFX — DaVinci 自家高品質插件，調色、特效一應俱全
[/flow]

### 穩定化（Stabilization）實戰

```prompt [label="穩定化操作步驟"]
1. 選取需要穩定的素材
2. Inspector（檢查器）→ 穩定化
3. 模式選擇：
   透視（Perspective）：最強，適合嚴重晃動
   相似性（Similarity）：平衡效果與裁切量
   移動（Translation）：僅補正位移，保留搖鏡感
4. 設定裁切比率：越高越穩，但邊框越多
5. 點「穩定化」→ 等待分析完成
```

# 模組 5：調色技術
> 調色是影片的「化妝」——相同的素材，調色不同可以說出截然不同的故事

## 5-1 色彩理論與示波器（1 小時）

### 影片色彩基礎

[flow]
1. RGB 色彩模型 — 紅綠藍三通道疊加，每通道 0-255（8bit）或更多
2. YCbCr 色彩空間 — Y（亮度）+ Cb（藍色差）+ Cr（紅色差），廣播標準
3. 色彩深度（Bit Depth）— 8bit（256色階）/ 10bit（1024色階）/ 12bit
4. 色域（Color Gamut）— Rec.709（螢幕標準）/ DCI-P3（電影院）/ Rec.2020（HDR）
5. Gamma 曲線 — 控制亮度與視覺感知的非線性關係
[/flow]

### 四種示波器詳解

[tags]
- [blue] Waveform（波形圖）：左到右反映畫面亮度分布，過曝超過 100%，欠曝低於 0%
- [orange] Parade（分量圖）：RGB 三通道獨立顯示，三線等高表示中性色（無色偏）
- [green] Vectorscope（向量圖）：顯示色相與飽和度，膚色應落在「膚色線」附近
- [purple] Histogram（直方圖）：整體亮度像素分布，山形偏左=欠曝，偏右=過曝
[/tags]

## 5-2 一級調色：讓畫面「正確」（1 小時）

### 一級調色的三個目標

一級調色（Primary Grading）解決的是技術問題，不是風格問題：

[flow]
1. 曝光校正 — 讓波形圖主體落在 10-90% 之間
2. 白平衡校正 — 讓 Parade 圖的 RGB 三條線等高
3. 對比度校正 — 確保暗部有細節、高光不過曝
[/flow]

[quiz type="bool"]
Q: 一級調色主要解決的是風格問題，而不是技術問題。
- [ ] 是
- [x] 否
Hint: 一級調色（Primary Grading）解決的是曝光、白平衡、對比度等技術問題，風格問題留給二級調色處理。
[/quiz]

```prompt [label="系統性一級調色流程"]
步驟 1：檢查 Waveform
  → 若整體偏高（過曝）：降低 Gain
  → 若整體偏低（欠曝）：提高 Lift

步驟 2：檢查 Parade
  → 若 B 通道偏高（偏藍）：在 Offset 微調往橘色方向
  → 若 R 通道偏高（偏紅）：在 Offset 微調往青色方向

步驟 3：調整對比度
  → Color Wheels → Contrast 微調
  → 或直接在 Curves 拉 S 形曲線
```

### Color Wheels 四個控制點

[tags]
- [blue] Lift（暗部/陰影）：控制畫面最暗的區域
- [orange] Gamma（中間調）：控制畫面中等亮度區域，影響整體觀感最大
- [green] Gain（亮部/高光）：控制畫面最亮的區域
- [purple] Offset（全域）：整體平移所有亮度值，常用於快速校正色偏
[/tags]

## 5-3 二級調色：創造視覺風格（1.5 小時）

### 二級調色工具箱

二級調色（Secondary Grading）針對特定顏色或區域做精確調整：

[flow]
1. 限定器（Qualifier）— 用吸管選取特定顏色範圍做調整
2. Power Window — 繪製自訂形狀的遮罩區域
3. HSL 曲線（Hue vs Saturation 等）— 精確調整特定色相的飽和度、明度
4. 自動追蹤 — Power Window 自動跟隨移動中的主體
[/flow]

### 打造三種經典風格

```prompt [label="Orange & Teal（青橙色調）"]
適合：戶外風景、動作影片、人像攝影
特徵：膚色偏橘暖、陰影偏冷藍綠

步驟：
1. Lift：往 Teal（青）方向偏移
2. Gain：往 Orange（橘）方向偏移
3. 用限定器選取膚色，Hue vs Hue 往橘色推
4. 整體飽和度降至 55-70，對比度微增
```

```prompt [label="電影感（Cinematic Look）"]
適合：劇情短片、訪談、微電影
特徵：低飽和、高對比、暗部提亮（Lifted Blacks）

步驟：
1. Lift（陰影）微微提亮，不讓暗部完全死黑
2. Gain 微降，壓制過亮高光
3. 整體飽和度降至 40-60
4. 加入輕微暈影（Vignette）引導視覺中心
```

```prompt [label="清新日系（Japanese Film Look）"]
適合：旅行 Vlog、生活紀錄、美食影片
特徵：高亮、低對比、暖色調、柔和陰影

步驟：
1. 整體曝光提亮 +0.3 到 +0.5
2. Lift 大幅提亮，讓陰影接近中灰
3. Gain 偏暖黃色
4. 飽和度降至 60-70，加入輕微薄霧感
```

## 5-4 LUT 應用與調色預設（30 分鐘）

### 什麼是 LUT？

LUT（Look-Up Table）是預設的色彩轉換表，本質上是把「輸入色值」轉換成「輸出色值」：

[tags]
- [blue] 技術型 LUT：將 LOG 素材轉換為 Rec.709 顯示標準（必要步驟）
- [orange] 創意型 LUT：應用特定風格，如電影感、膠片感、復古風
- [green] 匹配型 LUT：讓不同攝影機拍攝的素材看起來一致
- [purple] 校正型 LUT：補償顯示器的色彩偏差
[/tags]

```prompt [label="套用 LUT 的正確順序"]
節點 1（技術型）：LOG → Rec.709 轉換 LUT
  ↓
節點 2（一級調色）：曝光、白平衡、對比度校正
  ↓
節點 3（創意型）：風格化 LUT，強度 30-70%
  ↓
節點 4（二級調色）：局部精調膚色、特定顏色
```

[callout type="warning" title="LUT 強度建議控制在 50% 以下"]
直接以 100% 強度套用創意 LUT，通常效果過於突兀。

在節點的「Opacity（不透明度）」調低到 30-70%，讓 LUT 的效果融合進你的一級調色，才是專業的使用方式。
[/callout]

# 模組 6：Fairlight 音頻後製
> 觀眾可以原諒畫質模糊，但絕對無法忍受爛音質——音頻是影片最容易忽視的關鍵

## 6-1 音頻基礎與軌道設定（45 分鐘）

### 音頻軌道架構

```prompt [label="標準音頻軌道設定"]
軌道 1：對白（Dialogue）  - 主要人聲
軌道 2：環境音（Ambience）- 場景背景音
軌道 3：音效（SFX）       - 動作音效、強調音效
軌道 4：音樂（Music）     - 背景音樂
── Bus ──
Mix Bus：所有軌道匯集輸出
```

### 音頻電平標準

[flow]
1. 對白目標電平 — -12 至 -6 dBFS，峰值不超過 -3 dBFS
2. 音樂電平 — 比對白低 15-20 dB，在對白出現時可動態降低
3. 音效電平 — 根據情境，強調音效可接近對白電平
4. 最終輸出 — 整體 LUFS 值：YouTube -14、廣播 -23
[/flow]

## 6-2 人聲處理三部曲（1 小時）

### 降噪（Noise Reduction）

```prompt [label="Fairlight 降噪操作步驟"]
方法一：取樣降噪
1. 找一段只有噪音（沒有說話）的片段，約 0.5-1 秒
2. 選取這段音頻 → 右鍵 → 「分析音頻」→「雜訊輪廓」
3. 在效果庫找「Voice Isolation」或「降噪」
4. 載入剛才的雜訊輪廓
5. 降噪強度：從 40-60% 開始，避免人聲失真

方法二：AI 人聲分離（Studio 版）
→ 自動將人聲與背景分離，效果更好但需要 Studio 版
```

### 均衡器（EQ）設定

三段式 EQ 對人聲的基本處理：

[tags]
- [blue] 80-120Hz 高通濾波：切除低頻噪音（冷氣聲、場地共鳴）
- [orange] 200-500Hz 微衰減：去除「悶」的中低頻，讓聲音更清晰
- [green] 2-4kHz 提升：增加人聲的「存在感」與清晰度
- [purple] 8-12kHz 微提升：增加空氣感，讓聲音更通透自然
[/tags]

### 壓縮器（Compressor）設定

```prompt [label="人聲壓縮建議參數"]
Threshold（閾值）：-18 至 -12 dBFS
  → 超過此音量才開始壓縮

Ratio（壓縮比）：3:1 至 4:1
  → 超出 3dB 壓縮成 1dB 輸出

Attack（起音時間）：5-10ms
  → 太快會壓掉子音（s、t、p 等）

Release（釋音時間）：60-100ms
  → 太短產生泵浦聲（pumping），太長壓不住

Make-up Gain：補回 3-5 dB
  → 補償壓縮損失的整體音量
```

[quiz type="single"]
Q: YouTube 影片的推薦音頻 LUFS 值是多少？
- [ ] -6 dB
- [ ] -23 dB
- [x] -14 dB
- [ ] -30 dB
Hint: 廣播電視標準是 -23 LUFS，YouTube 則使用較寬鬆的標準。
[/quiz]

## 6-3 音效設計與音樂混音（45 分鐘）

### 音效的作用層次

[flow]
1. 同步音效（Sync SFX）— 與畫面完全同步的動作音（腳步聲、開門聲）
2. 情緒音效（Atmos SFX）— 烘托情緒的音效（緊張的心跳聲、場景環境音）
3. 設計音效（Design SFX）— 科幻、超自然等不存在的聲音，需創意設計
[/flow]

### 背景音樂混音技巧

```prompt [label="動態音量（Automation）設定"]
目標：對白出現時音樂自動降低，對白結束後恢復

步驟：
1. 在音樂軌道點擊「A」開啟自動化模式
2. 播放時在對白開始前 0.5 秒，手動降低音量推桿
3. 在對白結束後 0.5 秒，恢復音量
4. DaVinci 自動記錄所有推桿動作為關鍵影格
5. 之後可在時間軸的音量包絡線直接調整
```

[callout type="tip" title="免版權音樂的合法來源"]
常見錯誤：從 YouTube 抓取背景音樂使用，即使是「免費分享」的音樂也可能有版權問題。

合法免費來源：YouTube Audio Library（YouTube 直接提供）、Pixabay Music、Free Music Archive。付費但版稅免除（Royalty-free）：Artlist、Epidemic Sound，適合商業影片。
[/callout]

# 模組 7：Fusion 特效入門
> 節點式特效看起來複雜，但掌握核心邏輯後，你能做到的比你想象的多得多

## 7-1 Fusion 節點系統（45 分鐘）

### 節點邏輯：把效果串起來

Fusion 使用節點（Node）而非圖層（Layer），每個節點做一件事：

```prompt [label="Fusion 基礎節點類型"]
MediaIn  → 輸入來源（你的影片素材）
Transform → 縮放、位移、旋轉
Merge    → 合併兩個元素（相當於 Photoshop 的圖層）
Text+    → 文字生成器
Background → 產生純色或漸層背景
MediaOut → 輸出到時間軸
```

### Fusion 基礎操作

[flow]
1. 從 Edit 頁面選取素材 → 點擊底部「Fusion」頁面按鈕
2. 素材自動以 MediaIn 節點出現在節點編輯器
3. 右鍵空白區域或使用 Shift+Space 新增節點
4. 拖動節點的輸出口（小三角）連接到下一個節點的輸入口
5. 點擊節點旁的「1」或「2」按鈕，在左/右監視器預覽該節點輸出
[/flow]

## 7-2 動態字幕與圖形製作（45 分鐘）

### 製作動態標題文字

```prompt [label="動態標題製作步驟"]
1. 在 Edit 頁面新增一個「Fusion Composition」到時間軸
2. 進入 Fusion 頁面 → 新增「Text+」節點
3. 連接：Text+ → Merge → MediaOut
4. Inspector 設定文字內容、字體、大小、顏色
5. 新增動畫：點選 Inspector 中要動畫的屬性 → 右鍵 → 「動畫」
6. 移動時間軸到不同位置，調整屬性值，自動產生關鍵影格
```

### 常用動態效果

[tags]
- [blue] 淡入（Fade In）：Text+ 的 Opacity（不透明度）從 0 → 1
- [orange] 位移出現：Transform 節點的 Y 軸從 -0.1 → 0，同時淡入
- [green] 縮放強調：Scale 從 1.0 → 1.05 → 1.0 的彈跳效果
- [purple] 打字機效果：Text+ 的 End 屬性從 0 → 1，字母逐個顯示
[/tags]

[callout type="tip" title="Fusion 特效的學習路徑建議"]
Fusion 的功能非常深，完整學習需要額外的專門課程。本模組的目標是讓你能製作常用的動態字幕和簡單圖形。

進階方向：學習 Fusion 的粒子系統、3D 合成、VFX 追蹤等功能，可以參考 Blackmagic Design 官方的免費認證課程。
[/callout]

## 8-3 常見問題 FAQ

[reveal title="免費版可以輸出 4K 影片嗎？"]
可以。DaVinci Resolve 免費版支援最高 Ultra HD（3840×2160）輸出，包含 H.264 和 H.265 格式。只有超過 4K 的解析度（如 8K）或某些專業格式（DNxHR、ProRes）才需要 Studio 版。
[/reveal]

[reveal title="DaVinci Resolve 和 Premiere Pro 哪個比較好學？"]
這取決於你的背景。如果你已經熟悉 Adobe 生態系統，Premiere Pro 的介面會比較親切。但 DaVinci Resolve 的頁面分工更清晰（Cut 給新手、Edit 給專業），而且免費版功能已經非常完整。對於完全從零開始的新手，DaVinci Resolve 的學習曲線其實更平緩。
[/reveal]

[reveal title="為什麼我的影片在 DaVinci 裡很卡？"]
最常見的原因是沒有使用代理媒體（Proxy）。4K 素材對大多數電腦來說都太重，建議：
1. 右鍵素材 → 生成代理媒體
2. 播放時切換為 Proxy 模式
3. 調色和輸出時再切換回原始素材

另外，確認 GPU 加速已在專案設定中啟用。
[/reveal]

[accordion]
[item title="電腦規格不夠，可以用 DaVinci Resolve 嗎？" open]
最低 16GB RAM 可以跑 1080p 剪輯。如果只有 8GB，建議：
- 使用代理媒體工作流程
- 關閉背景程式
- 在偏好設定中降低播放解析度到 1/4

升級 RAM 到 32GB 是剪輯 4K 最值得的投資。
[/item]
[item title="如何從 Premiere Pro 轉移專案到 DaVinci？"]
可以使用 EDL 或 XML 格式交換剪輯資料：
1. 在 Premiere 中匯出 XML / EDL
2. 在 DaVinci 的媒體資料庫中載入該檔案
3. 時間軸會自動重建（轉場和效果需要重新套用）

注意：調色、Fusion 特效、Fairlight 混音無法轉移，需重新製作。
