# 概覽：兩套工具，一套工作流程
> Photopea 修圖，Vectorpea 排版——搞清楚分工，才知道每一步要去哪裡操作

## 傳統印前 vs AI 印前

[compare label-left="傳統印前流程" label-right="AI 印前流程"]
- 專業設計師用 Photoshop/Illustrator 手動製作 | AI 生成圖片後，用 Photopea/Vectorpea 修復排版
- 素材來自攝影棚拍攝或客戶提供 | 素材來自 Midjourney、DALL-E 等 AI 生圖工具
- 原始檔案品質可控、解析度足夠 | AI 圖片通常 72dpi，需檢查解析度並修復視覺缺陷
- 色彩管理流程成熟，直接輸出 CMYK | AI 圖片為 RGB，需手動轉換並收斂飽和度
- 印前檢查靠經驗與人工 | 可用系統化清單（本課程提供）逐项確認
[/compare]

## 點陣圖與向量圖的根本差異

### 兩種截然不同的圖形語言

[compare label-left="點陣圖（Raster）" label-right="向量圖（Vector）"]
- 由固定像素組成，放大超過原始尺寸會模糊 | 由數學公式描述路徑和形狀，無限縮放不失真
- AI 生圖工具（Midjourney、Stable Diffusion、DALL-E）輸出皆為點陣 | Logo、印刷用字體、插圖輪廓應為向量
- 格式：JPG / PNG / WEBP / PSD | 格式：SVG / PDF / AI / EPS
- 適合：照片、寫實場景、漸層豐富圖片 | 適合：Logo、圖示、文字排版、幾何圖形
[/compare]

[tags]
- [orange] 點陣圖 JPG/PNG：像素固定，放大就糊，AI 生圖的預設輸出
- [green] 向量圖 SVG/PDF/AI：公式計算，無限縮放，Logo 與排版的標準格式
- [blue] 印刷原則：照片類用高解析度點陣，Logo 與文字用向量
[/tags]

## Photopea 與 Vectorpea 的分工

### 各司其職，互相補充

**Photopea**（photopea.com）：免費瀏覽器版 Photoshop，處理點陣圖。用來修復 AI 圖片的瑕疵、調整色彩和解析度、去背合成，以及進行 CMYK 色彩轉換。

**Vectorpea**（vectorpea.com）：免費瀏覽器版 Illustrator，處理向量圖。用來編輯 SVG 和 AI 格式、排版文字、繪製向量圖形，以及輸出印刷用的向量 PDF。

兩套工具均由同一位開發者（Ivan Kutskir）開發，介面邏輯一致，學完一套，另一套上手更快。

## 完整工作流程總覽

### 從 AI 生圖到送印的每一步

[flow]
1. AI 工具生圖 — Midjourney、Stable Diffusion 等，輸出 JPG/PNG
2. Photopea：診斷 — 檢查解析度、色彩模式、AI 圖片常見缺陷
3. Photopea：修復 — 修復手指、去除文字圖形、處理邊緣光暈
4. Photopea：調色 — 曲線、色相飽和度、色彩平衡，符合印刷色域
5. Vectorpea：生成向量元素 — AI 生成 SVG、向量化點陣圖或手動描圖
6. Vectorpea：排版 — 放入圖片、加上文字、對齊版面
7. 規格確認 — 尺寸、出血、安全區域、色彩模式
8. 輸出送印 — Photopea 匯出 TIFF/PSD，Vectorpea 匯出 PDF
[/flow]

### 哪些專案只需要 Photopea？哪些需要兩者並用？

[callout type="warning" title="印前三大致命錯誤"]
- **出血未留**：設計尺寸未加 3mm 出血，裁切後邊緣露白邊，整批重印
- **色彩模式錯誤**：未轉 CMYK 就送印，鮮豔色印出來暗沉偏色，客戶拒收
- **解析度不足**：72dpi 圖片直接放大印刷，馬賽克明顯，近看完全無法接受
[/callout]

| 專案類型 | 工具需求 |
|----------|---------|
| AI 生圖修圖後直接送印（海報、相紙） | 僅 Photopea |
| 有 Logo 和文字排版的印刷品（傳單、名片） | Photopea + Vectorpea |
| 純向量設計（Logo、圖示、插圖） | 僅 Vectorpea |
| AI 圖片 + 向量排版整合（完整設計稿） | Photopea + Vectorpea |

# Photopea 入門：介面與圖層基礎
> 不需要安裝，開啟瀏覽器就能用——但要用得好，先把介面和圖層搞清楚

## 工作介面導覽

### 四大區域

[flow]
1. 頂部選單列 — File、Edit、Image、Layer、Select、Filter 等功能選單
2. 左側工具列 — 選取、裁切、修復、筆刷、文字等工具（快速鍵與 Photoshop 相同）
3. 中央畫布 — 影像編輯的主要工作區域
4. 右側面板 — 圖層（Layers）、色版（Channels）、屬性（Properties）
[/flow]

**圖層面板**：是整個後處理流程的核心控制中心，Window → Layers 可開啟。**屬性面板**：選取調整圖層後，在此修改曲線、色相飽和度等參數。

### 開啟 AI 生成圖片

[flow]
1. 直接拖放 — 將圖片從桌面或資料夾拖入 Photopea 視窗（最快）
2. File → Open — 從電腦選取 JPG、PNG、WEBP、PSD 等格式
3. File → Open from URL — 輸入圖片網址直接開啟
[/flow]

圖片開啟後，立刻到 **Image → Image Size** 查看寬度、高度、解析度三個數字，這三個數值決定後續的處理策略。

### 基本導航

| 操作 | 快速鍵 |
|------|--------|
| 放大 | Ctrl + + |
| 縮小 | Ctrl + - |
| 符合視窗 | Ctrl + 0 |
| 實際大小 | Ctrl + 1 |
| 平移畫布 | 按住空白鍵拖移 |

Ctrl + Z 還原一步，Ctrl + Alt + Z 連續還原多步。

## 圖層觀念與無損編輯

### 透明膠片的比喻

把圖層想像成一疊透明膠片，每張膠片有不同內容，疊在一起就是最終畫面。可以單獨調整某一張，不影響其他層。

[flow]
1. 最底層 — AI 生成的原始圖片（背景圖層）
2. 中間層 — 調整圖層（曲線、色相等，不破壞像素）
3. 上方層 — 修復筆刷的操作結果（放在獨立空白圖層）
4. 最上層 — 文字或 Logo 等設計元素
[/flow]

> **第一條守則：永遠不直接編輯原始圖層**
> 開始任何操作前，按 Ctrl + J 複製圖層，在複本上操作。無論發生什麼，原始圖片都還在。

### 三種關鍵圖層類型

**像素圖層**：儲存實際像素資料，AI 生成的圖片開啟後就是這個類型。

**調整圖層**：不儲存像素，只儲存調整參數，對下方所有圖層產生效果，隨時可修改。在圖層面板底部的半圓圖示建立。

[tags]
- [green] 調整圖層：曲線、色相飽和度、色彩平衡都建議用這個
- [green] 可用遮罩限制調整範圍（只調背景，不影響主體）
- [orange] 不能在調整圖層上直接用筆刷繪畫
[/tags]

**智慧物件**：封裝圖層後縮放不損失畫質，右鍵圖層 → Convert to Smart Object。AI 圖片需要縮放時建議先轉換。

### 必學的六個圖層操作

[flow]
1. 複製圖層 — Ctrl + J
2. 新增空白圖層 — Ctrl + Shift + N（修復工具的操作空間）
3. 合併圖層 — Ctrl + E（選取的圖層），Ctrl + Shift + E（全部可見）
4. 圖層不透明度 — 面板右上角 Opacity 數值
5. 圖層混合模式 — 不透明度旁的下拉選單
6. 圖層遮罩 — 面板底部的方框圖示
[/flow]

# 修復：AI 圖片診斷與瑕疵清除
> 先做「健康檢查」，再對症下藥——這個順序不能顛倒

## 診斷 AI 圖片的三個問題

### 問題一：解析度不足

dpi（dots per inch）是每英寸像素點數。螢幕只需 72–96dpi，印刷需要至少 **300dpi**，否則放大後出現明顯馬賽克。

| AI 圖片像素 | 300dpi 可印最大尺寸 | 適合用途 |
|-------------|-------------------|---------|
| 1024 × 1024 | 8.7 × 8.7 公分 | 名片、小貼紙 |
| 2048 × 2048 | 17.4 × 17.4 公分 | A5 以下 |
| 4096 × 4096 | 34.7 × 34.7 公分 | A3 左右 |

**檢查方式**：Image → Image Size，查看 Resolution 欄位。

### 問題二：色彩模式（RGB vs CMYK）

螢幕用 RGB（光的三原色）發光，色域廣，支援霓虹色。印刷用 CMYK（四色油墨）疊印，色域較窄，超出範圍的鮮豔色轉換後會明顯偏暗。

**檢查方式**：Image → Mode，目前模式會打勾。AI 圖片幾乎都是 RGB，送印前視廠商要求轉換。

### 問題三：AI 視覺缺陷

[tags]
- [orange] 高風險：手指數量錯誤或關節扭曲、AI 生成的「文字圖形」（無意義）
- [orange] 中風險：主體邊緣光暈或鋸齒、背景比例透視錯誤
- [blue] 低風險：局部色彩偏差、細節模糊
[/tags]

**快速診斷流程**：

[flow]
1. Ctrl + 1 縮放到 100% 實際大小，全圖瀏覽一遍
2. 特別放大人物手部區域（Ctrl + + 放大，空白鍵平移）
3. 確認圖中是否有「看起來像字」的 AI 生成文字圖形需要覆蓋
4. 觀察主體邊緣是否有光暈或鋸齒
5. 記錄問題區域，開始修復
[/flow]

## 三種修復工具

[tags]
- [green] 污點修復筆刷（J）：小範圍瑕疵，自動分析周邊紋理
- [blue] 仿製印章（S）：精確複製來源，修復手指、移除元素
- [orange] 修補工具（Shift+J）：大面積置換，移除人物、替換背景
[/tags]

### 污點修復筆刷（J）

**適用**：小範圍瑕疵——臉上斑點、背景雜訊點。自動分析周邊紋理填補，不需指定來源。

[flow]
1. Ctrl + J 複製圖層，在複本上操作
2. 按 J 選取污點修復筆刷
3. 調整筆刷大小略大於瑕疵區域
4. 直接點擊或塗抹瑕疵，自動填補
[/flow]

> **在空白圖層上操作更安全**：新增空白圖層（Ctrl + Shift + N），勾選選項列的「Sample All Layers」，修復結果在獨立圖層上，不滿意可直接刪除。

### 仿製印章（S）

**適用**：需要精確複製某區域覆蓋問題——修復 AI 手指、移除多餘元素。需手動指定「來源點」。

[flow]
1. 按 S 選取仿製印章
2. 調整筆刷大小與不透明度
3. 按住 Alt，點選要複製的乾淨來源區域
4. 放開 Alt，在問題區域塗抹覆蓋
[/flow]

**修復 AI 手指策略**：放大到 200–400%，從相鄰正確手指複製紋理，覆蓋多餘或錯誤的手指，邊緣用低不透明度筆刷做羽化過渡。

### 修補工具（Shift + J 切換）

**適用**：大面積置換——移除人物、替換大塊背景。先框選問題區域，再拖移到乾淨的來源區域。

[flow]
1. 用修補工具圈選要移除的問題區域
2. 將選取範圍拖移到旁邊紋理相似的乾淨區域
3. 放開滑鼠，Photopea 自動混合邊緣
4. Ctrl + D 取消選取查看結果
[/flow]

**AI 文字的處理**：用仿製印章或修補工具覆蓋 AI 生成的「文字圖形」，再用文字工具（T）重新輸入正確文字。

## 選取與去背

### 依對比度選擇工具

[flow]
1. 主體與背景顏色反差大 → 魔術棒（W）或顏色範圍（Select → Color Range）
2. 主體輪廓清晰但不規則 → 快速選取工具（W）
3. 主體有細碎邊緣（頭髮、毛皮）→ Select and Mask 邊緣細化
4. 需要精確手動控制 → 鋼筆工具（路徑）
[/flow]

### Select and Mask 邊緣細化

進入 Select → Select and Mask 後，關鍵設定：

**Smart Radius**：勾選後自動判斷軟硬邊，建議預設開啟。**Radius**：頭髮邊緣設 10–15px，硬邊物件設 1–3px。**Feather**：0.5–1px 讓邊緣稍微柔化，合成更自然。

[flow]
1. 快速選取工具大略選出主體
2. 進入 Select and Mask
3. 開啟 Smart Radius，調整 Radius
4. 用邊緣細化筆刷塗抹頭髮等細碎邊緣
5. Output 設為「New Layer with Layer Mask」
[/flow]

> **守則：去背一律使用圖層遮罩，不用橡皮擦**
> 遮罩可隨時修改，橡皮擦直接刪除像素不可逆。客戶要求微調邊緣時，遮罩能讓你快速應對。

# 調色：讓 AI 圖片符合印刷色彩
> AI 圖片針對螢幕優化，通常飽和度過高——印刷前需要收斂，才能忠實還原

## 調整圖層的正確使用方式

永遠用 **Layer → New Adjustment Layer**（或圖層面板底部半圓圖示），不用 Image → Adjustments 選單。後者直接作用在像素上不可逆，前者隨時可修改參數。

## 四種必學調整

### 亮度/對比（基礎快速）

建立「亮度/對比」調整圖層，快速修正整體曝光。

[tags]
- [green] Brightness 正值提亮，負值壓暗
- [blue] Contrast 正值增強對比，負值降低
- [orange] 印刷建議：對比不要設太高，陰影部分油墨量容易過重
[/tags]

### 曲線（精準控制）

最精細的色調控制工具。橫軸是原始亮度，縱軸是調整後亮度。

**S 型曲線**：同時提亮高光、壓暗陰影，增加立體感，是最常用的基本調整。

**針對印刷的曲線策略**：
- 高光部分（右上角）略微壓低，避免白色區域印刷時看不到細節
- 陰影部分（左下角）略微提高，避免深色區域油墨疊印過重

在下拉選單切換到 Red、Green、Blue 頻道，可修正偏色問題（例如圖片偏黃：在 Blue 頻道高光部位略微提高）。

### 色相/飽和度（AI 圖必做）

Midjourney 等 AI 工具的圖片飽和度通常偏高，轉 CMYK 後鮮豔色彩會被壓縮，印出來暗沉。送印前在飽和度上拉 -10 到 -20 讓色彩稍微收斂。

**針對特定顏色**：下拉選單由「Master」切換到「Reds」、「Blues」等，單獨調整過飽和的天空藍或膚色。

### 色彩平衡（修正偏色）

分別對 Shadows、Midtones、Highlights 做色調調整。AI 室內場景常偏橘黃，如需冷色調，在中間調略增 Cyan 和 Blue。

## RGB 轉 CMYK

### 轉換時機與步驟

轉換應在所有調色工作完成之後——轉換後色域縮小，部分顏色會變化，無法回頭。

[flow]
1. 確認所有調整圖層效果滿意
2. Image → Mode → CMYK Color
3. Photopea 提示合併調整圖層，選「OK」
4. 觀察顏色變化，特別注意明顯偏色區域
5. 如有需要，在 CMYK 模式下再用色彩平衡微調
[/flow]

在轉換前可開啟 **View → Gamut Warning**，超出 CMYK 色域的顏色會顯示灰色遮罩，提前知道哪裡會有問題。

> **注意：先詢問印刷廠**
> 有些廠商接受 RGB（他們自行轉換），有些要求 CMYK。按廠商規格操作，避免重工。

[reveal title="為什麼 AI 圖片轉 CMYK 後會變暗沉？"]
螢幕 RGB 使用「加色法」發光，色域涵蓋霓虹色等高飽和色彩。印刷 CMYK 使用「減色法」油墨疊印，色域較窄。當鮮豔的 RGB 色彩超出 CMYK 可呈現範圍時，系統會強制壓縮到最近的可印刷色，結果就是明顯變暗、失去原有的鮮豔感。送印前把飽和度拉低 -10 到 -20，可以讓轉換後的結果更接近螢幕預期的效果。
[/reveal]

# Vectorpea 入門：向量圖形基礎
> 向量圖不是「更清晰的 JPEG」，而是一種由路徑和公式組成的圖形語言

## 工作介面導覽

### 四大區域

[flow]
1. 頂部選單列 — File、Edit、Object、Text、View 等功能選單
2. 左側工具列 — 選取、節點編輯、鋼筆、形狀、文字等工具
3. 中央畫布 — 向量編輯工作區，白色矩形是工作畫板
4. 右側屬性面板 — 填色、筆畫、字型、對齊等控制項
[/flow]

### 最常用工具快速鍵

| 工具 | 快速鍵 |
|------|--------|
| 選取工具 | V |
| 節點編輯 | N |
| 鋼筆工具 | P |
| 文字工具 | T |
| 矩形工具 | R |
| 橢圓工具 | E |

## 路徑、節點與錨點

### 向量的基本單位

所有向量圖形的基礎都是**路徑（Path）**：由錨點連接而成的線段或曲線。

**封閉路徑**：起點與終點相連，形成可填色的形狀（圓形、矩形、字母輪廓）。
**開放路徑**：起點與終點不相連，只能設定筆畫（stroke），不能填色。

錨點有兩種：
- **尖角錨點（Corner）**：路徑在此急轉，用於銳利角形狀
- **平滑錨點（Smooth）**：兩側控制把手在同一直線，用於曲線和弧形

## 填色（Fill）與筆畫（Stroke）

每個向量形狀都有兩層視覺屬性，在右側面板控制：

**Fill（填色）**：封閉路徑內部的顏色，可以是純色、漸層或圖樣。
**Stroke（筆畫）**：沿路徑邊緣繪製的線條，可設定顏色、寬度和端點樣式。

[tags]
- [green] Fill：路徑內部，主體色彩
- [blue] Stroke：路徑外框線，可設粗細和端點樣式
- [orange] 印刷重點：筆畫不小於 0.25pt，過細印刷時可能斷線消失
[/tags]

## 圖層管理

Vectorpea 的圖層概念與 Photopea 相似，但每個向量物件都是獨立可選取的。建議圖層組織：

[flow]
1. 最上層 — 文字圖層（確保不被遮蓋）
2. 中間層 — 裝飾性向量圖形（插圖、圖示）
3. 底層 — 背景色塊或置入的點陣圖片
[/flow]

# AI 生成 SVG 與路徑編輯
> AI 生圖不只是 JPG/PNG——善用正確工具，可以直接生成可編輯的向量圖形

## 三種取得 SVG 的方法

### 方法一：請 AI 直接生成 SVG 代碼

ChatGPT、Claude 等 AI 助理可以直接生成 SVG 代碼。SVG 是純文字格式，複製貼上到文字編輯器存成 `.svg` 後，即可用 Vectorpea 開啟編輯。

**適合的圖形類型**：幾何圖形、簡單插圖、圖示（Icon）、資訊圖表、線條藝術。不適合寫實照片或複雜細節圖形。

```prompt label="向 AI 要求 SVG 的提示詞範例"
請生成一個 SVG 圖形：咖啡杯的簡單插圖，
使用純色填充，不要漸層，大小 200x200px，適合用作 Logo。
請直接輸出 SVG 代碼，不要解釋。
```

[flow]
1. 複製 AI 生成的 SVG 代碼
2. 開啟文字編輯器，貼上代碼，存成 logo.svg
3. Vectorpea → File → Open，選取 logo.svg
4. 所有路徑以可編輯向量物件匯入
[/flow]

### 方法二：AI 向量化工具將點陣圖轉 SVG

把 AI 生成的 JPG/PNG 上傳到向量化工具自動描圖，轉換成可在 Vectorpea 編輯的 SVG。

**推薦工具**：Recraft.ai（AI 驅動，路徑較乾淨）、Adobe Express 向量化（與 Illustrator 相容）。

[tags]
- [green] 適合：高對比度、邊緣清晰、顏色數量少的插圖風格圖片
- [green] 適合：Logo、圖示、剪影、幾何圖案
- [orange] 不適合：寫實照片、細碎紋理、漸層豐富的圖片
[/tags]

[flow]
1. 準備 AI 生成的 JPG/PNG（白色或透明背景效果最好）
2. 上傳到向量化工具，下載輸出的 SVG
3. Vectorpea → File → Open 開啟 SVG
4. 整理路徑（見下方節點編輯章節）
[/flow]

### 方法三：手動描圖

以 AI 圖片作為底圖，用鋼筆工具在上方手動描繪路徑。最費時，但結果最乾淨，適合 Logo 設計。

[flow]
1. File → Open 開啟 AI 圖片（JPG/PNG）
2. 圖層面板中鎖定底圖圖層（避免誤選）
3. 新增一個圖層，在上方操作
4. 鋼筆工具（P）沿圖片輪廓描繪路徑
5. 完成後隱藏或刪除底圖圖層
[/flow]

## 節點編輯與路徑整理

### 進入節點編輯模式

選取向量物件後按 N，或直接雙擊物件。自動向量化後通常有過多節點，整理目標是：**用最少節點描述最平滑的路徑**。

| 操作 | 方法 |
|------|------|
| 選取節點 | 點擊或框選錨點 |
| 移動節點 | 拖移錨點 |
| 刪除節點 | 選取後按 Delete（路徑自動重連） |
| 新增節點 | 在路徑段上點擊 |
| 切換尖角/平滑 | 雙擊錨點 |

[flow]
1. 進入節點編輯模式（N）
2. 找出不必要的節點（直線段中間、方向相同的連續節點）
3. 逐一刪除多餘節點，確認路徑形狀維持
4. 調整控制把手讓曲線更平滑
5. 檢查開放路徑：Object → Path → Close Path 封閉斷開的路徑
[/flow]

### 布林運算整理色塊

**Unite（聯集）**：合併分離的相同色塊，減少路徑數量，避免印刷時出現色塊間的白縫（漏白問題）。

**Subtract（差集）**：用上方形狀裁切下方，製作鏤空效果（字母的負空間、Logo 的透空區域）。

在 Object → Path Operations 找到以上選項。

# 排版：Vectorpea 版面設計
> 在 Vectorpea 裡完成文字排版、形狀設計、版面對齊，一個檔案搞定整張設計稿

## 文字工具

### 兩種文字建立方式

**點擊文字（Point Text）**：按 T 後點擊畫布輸入，不會自動換行，適合單行標題。

**區域文字（Area Text）**：按 T 後拖曳出矩形框，在框內輸入，超出框的文字自動隱藏，適合段落正文。

### 字型設定

右側面板 Character 區域：
- **Font Family**：字體選擇，中文設計用系統內建繁體中文字型
- **Font Size**：以 pt 為單位，印刷正文約 9–12pt
- **Line Height**：建議字級的 1.4–1.6 倍（12pt 字配 17–20pt 行距）
- **Letter Spacing**：標題可適度增加字距，正文保持預設

## 文字轉外框（送印必做）

文字轉外框將文字圖層轉為向量路徑，印刷廠不需安裝你的字型就能正確輸出。

[flow]
1. 確認文字內容和排版定稿（轉外框後無法再編輯文字）
2. 先 File → Save 存一份帶文字的 SVG 工作檔備份
3. 選取所有文字物件
4. Text → Create Outlines（Shift + Ctrl + O）
5. 再存一份送印版本，命名加「_outline」以區分
[/flow]

> **轉外框是單向操作**：備份工作檔這個步驟不能省，客戶要改文字時，只有工作檔救得了你。

## 形狀工具與對齊

### 基本形狀

- **矩形（R）**：按 Shift 繪製正方形
- **橢圓（E）**：按 Shift 繪製正圓
- **多邊形**：可設定邊數，繪製三角形、五角形等

形狀建立後可用 Object → Convert to Path 轉換為可任意編輯的路徑。

### 對齊與分佈

[flow]
1. Shift + 點擊多選需要對齊的物件
2. 選擇對齊基準（對齊到選取範圍、對齊到畫板）
3. 點擊對應的對齊按鈕（左對齊、置中、右對齊等）
4. 用水平/垂直分佈讓物件間距相等
[/flow]

## 版面設計原則

**視覺層次**：標題字級 > 副標題 > 正文；顏色對比明顯；留白讓版面呼吸。

**出血（Bleed）**：設計尺寸四邊各加 3mm，確保裁切誤差不露白邊。

**安全區域**：重要內容（文字、Logo）距離設計邊緣至少留 3–5mm。

# 規格與輸出：送印前的最終準備
> 最後一關——尺寸、出血、色彩模式、檔案格式，四件事缺一不可

## 印刷尺寸規格

### 常見台灣印刷品尺寸

| 印刷品 | 完成尺寸（mm） | 含出血尺寸（mm） | 300dpi 像素需求 |
|--------|--------------|----------------|----------------|
| 名片 | 90 × 54 | 96 × 60 | 1134 × 709 |
| A5 傳單 | 148 × 210 | 154 × 216 | 1819 × 2551 |
| A4 傳單 | 210 × 297 | 216 × 303 | 2551 × 3508 |
| A3 海報 | 297 × 420 | 303 × 426 | 3508 × 4961 |

**大型輸出（展覽看板、戶外廣告）**：觀看距離遠，通常允許 150dpi，不需達到 300dpi。

## 出血設定

### Photopea 中設定出血

[flow]
1. Image → Canvas Size（不是 Image Size）
2. 寬度 + 0.6cm（左右各 3mm），高度 + 0.6cm（上下各 3mm）
3. Anchor 設定為中心點
4. 確認後確保背景圖片填滿擴大的畫布
[/flow]

### Vectorpea 中設定出血

[flow]
1. File → New Document，在尺寸欄直接輸入含出血的尺寸
2. 或在已有的文件中，File → Document Properties 調整畫板尺寸
3. 在頂部尺規拖出參考線，標記安全區域（距邊緣 3–5mm）
4. View → Lock Guides 鎖定參考線避免誤移
[/flow]

## Photopea 的解析度調整

### Image Size 的正確操作

**Resample 勾選**：Photopea 會補像素。印刷放大時選「Preserve Details 2.0」，縮小時選「Bicubic Sharper」。差距超過 120% 建議先用 AI 放大工具（Topaz Gigapixel、Magnific AI）再導入。

**Resample 不勾選**：只改標籤數字，實際像素不變，等於告訴印表機「用這個 dpi 解讀」，原圖像素不足還是會模糊。

## 匯出格式對照

### Photopea 匯出

| 格式 | 用途 | 操作路徑 |
|------|------|---------|
| PSD | 保留圖層的工作備份 | File → Save |
| PDF | 送印主格式 | File → Export As → PDF |
| JPEG | 校稿版本，供客戶確認 | File → Export As → JPEG（品質 85–90） |
| PNG | 含透明底的圖層 | File → Export As → PNG |

### Vectorpea 匯出

| 格式 | 用途 | 操作路徑 |
|------|------|---------|
| SVG | 工作備份（保留文字可編輯） | File → Save |
| PDF | 送印主格式（含向量路徑） | File → Export As → PDF |
| PNG | 校稿版本或社群媒體用 | File → Export As → PNG |

**PDF 送印注意**：Vectorpea 匯出 PDF 時確認勾選「Include bleed」（含出血輸出）；Photopea 匯出 PDF 時確認色彩模式已是 CMYK（若廠商要求）。

## 送印前最終檢查清單

[steps-status]
- [done] 解析度確認 | Photopea 中 Image → Image Size 確認 300dpi（大型輸出可 150dpi）
- [done] 色彩模式 | CMYK 已轉換或按廠商要求確認 RGB 可接受
- [done] AI 缺陷修復 | 手指、文字圖形、邊緣光暈已處理
- [done] 文字轉外框 | Vectorpea 中 Text → Create Outlines（已存工作備份）
- [done] 筆畫寬度 | 不小於 0.25pt，過細印刷會斷線
- [done] 向量路徑 | 全部封閉，相同色塊已合併避免漏白
- [doing] 出血設定 | 畫布/畫板每邊 +3mm，背景填滿
- [doing] 安全區域 | 重要元素距設計邊緣至少 3-5mm
- [todo] 匯出送印 | 匯出 PDF 給印刷廠，JPEG/PNG 給客戶校稿
[/steps-status]

> **每次送印前逐項確認**
> 印刷出錯的代價遠超過花幾分鐘做檢查。養成這個習慣，可以避免絕大多數的重印費用和客戶糾紛。
