# 入門：認識 Vectorpea 與向量圖的世界
> 向量圖不是「更清晰的 JPEG」，而是一種完全不同的圖形語言——理解這個差異，是學好 Vectorpea 的第一步

## 點陣圖 vs 向量圖

### 根本的差異

[compare label-left="點陣圖（Raster）" label-right="向量圖（Vector）"]
- 由固定數量的像素組成 | 由數學公式描述的路徑和形狀組成
- 放大超過原始尺寸就會模糊失真 | 放大到任何尺寸都不失真
- 常見格式：JPG、PNG、WEBP | 常見格式：SVG、AI、PDF
- AI 生圖工具預設輸出格式 | Logo、插圖、字體輪廓應使用此格式
- 印刷大尺寸需極高解析度 | 無限縮放，檔案大小與尺寸無關
[/compare]

[callout type="info" title="如何記憶這個差異"]
點陣圖就像**馬賽克拼貼**——每塊磁磚大小固定，拉開距離看就糊了。
向量圖就像**幾何公式**——不管畫多大，公式重新計算一次就對了。
[/callout]

**點陣圖（Raster）**：由固定數量的像素組成，放大超過原始尺寸就會模糊。AI 生圖工具（Midjourney、Stable Diffusion）輸出的 JPG/PNG，都是點陣圖。

**向量圖（Vector）**：由數學公式描述的路徑和形狀組成，放大到任何尺寸都不失真。Logo、插圖、字體輪廓，都應該是向量格式。

[tags]
- [orange] 點陣圖：JPG、PNG、WEBP——像素固定，放大就糊
- [green] 向量圖：SVG、AI、PDF——公式計算，無限縮放
- [blue] 印刷需求：大尺寸輸出必須用向量或超高解析度點陣
[/tags]

### 什麼時候需要向量圖？

[flow]
1. 名片、信封、信紙上的 Logo——需要小到 1cm 和大到 A3 都清晰
2. 戶外廣告、展覽看板——近看遠看都要清楚
3. 刺繡、雷射切割、CNC 加工——機器讀取路徑而非像素
4. 印刷品上的文字——字體必須轉外框，避免缺字問題
5. 客戶索取「可編輯的向量檔」——SVG 或 AI 格式
[/flow]

## 認識 Vectorpea

### 免費的瀏覽器版 Illustrator

Vectorpea 是 Photopea 作者 Ivan Kutskir 開發的向量圖形編輯器，介面與操作邏輯接近 Adobe Illustrator，完全在瀏覽器中運行，不需安裝。網址：vectorpea.com

**支援格式**：
- **開啟**：SVG、AI（Adobe Illustrator）、PDF
- **匯出**：SVG、PDF、PNG（點陣化輸出）

> **Vectorpea 與 Photopea 的分工**
> Photopea 處理點陣圖（修圖、合成、印刷解析度調整），Vectorpea 處理向量圖（路徑、排版、無限縮放）。兩者可以搭配使用：先在 Photopea 處理 AI 生成的點陣圖，再在 Vectorpea 做排版和向量整合。

[compare label-left="點陣圖編輯（Photopea）" label-right="向量圖編輯（Vectorpea）"]
- 以像素為單位操作（選取、塗抹、濾鏡） | 以物件為單位操作（路徑、錨點、形狀）
- 選取靠顏色或邊緣偵測 | 每個物件獨立可選取，不需選取範圍
- 放大後像素化，無法無限縮放 | 放大不失真，適合印刷大尺寸輸出
- 適合修圖、合成、照片後製 | 適合 Logo、排版、幾何圖形設計
[/compare]

# 概念：向量圖形的核心語言
> 路徑（Path）、節點（Node）、錨點（Anchor）——這三個概念是 Vectorpea 所有操作的基礎

## 路徑是什麼？

### 向量的基本單位

所有向量圖形的基礎都是**路徑（Path）**。路徑是由錨點連接而成的線段或曲線，可以是開放的（線條）或封閉的（形狀）。

**封閉路徑**：起點與終點相連，形成可以填色的形狀，例如圓形、矩形、字母輪廓。
**開放路徑**：起點與終點不相連，只能設定筆畫（stroke），不能填色。

### 錨點（Anchor Point）與控制把手

每個錨點定義了路徑的轉折點。錨點有兩種：

**尖角錨點（Corner）**：路徑在此急轉，兩側方向獨立控制。用於製作有銳利角的形狀。

**平滑錨點（Smooth）**：路徑在此平滑過渡，兩側的控制把手在同一直線上。用於製作曲線和弧形。

## 填色（Fill）與筆畫（Stroke）

### 向量圖形的兩層外觀

每一個向量形狀都有兩個視覺屬性，在右側面板控制：

**Fill（填色）**：封閉路徑內部的顏色，可以是純色、漸層或圖樣。開放路徑無法填色。

**Stroke（筆畫）**：沿著路徑邊緣繪製的線條，可以設定顏色、寬度和端點樣式。印刷時筆畫寬度至少設定 0.25pt，過細的線條在印刷機上可能消失。

[callout type="warning" title="印刷筆畫最小寬度"]
送印時筆畫寬度**不可低於 0.25pt**（約 0.088mm）。
螢幕上看得到不代表印得出來——印刷機的網點解析度有限，過細線條會斷線或完全消失。
[/callout]

[tags]
- [green] Fill：路徑內部的顏色，印刷主體色彩
- [blue] Stroke：路徑外框線，可設定粗細和樣式
- [orange] 印刷注意：筆畫過細（小於 0.25pt）印刷時可能斷線
[/tags]

## 圖層管理

### 與 Photoshop 圖層的差異

Vectorpea 的圖層概念與 Photopea 相似，但向量圖形的每個物件都是獨立可選取的，不像點陣圖那樣「合在一個圖層裡」。

建議的圖層組織方式：

[flow]
1. 最上層：文字圖層（確保文字不被其他物件遮蓋）
2. 中間層：裝飾性向量圖形（插圖、圖示）
3. 底層：背景色塊或背景圖片
[/flow]

# AI 生成：用三種方法產生 SVG 圖形
> AI 生圖不只是 JPG/PNG——善用正確工具，可以直接生成可編輯的向量 SVG

## 方法一：請 AI 直接生成 SVG 代碼

### 用文字提示生成向量圖形

ChatGPT、Claude 等 AI 助理可以直接生成 SVG 代碼。SVG 是純文字格式，可以直接複製貼上到文字編輯器並存成 `.svg` 檔，再用 Vectorpea 開啟編輯。

**適合的圖形類型**：幾何圖形、簡單插圖、圖示（Icon）、資訊圖表、線條藝術。不適合寫實照片風格或複雜細節圖形。

### 提示詞範例

```prompt label="向 AI 要求 SVG 的提示詞範例"
請生成一個 SVG 圖形：咖啡杯的簡單插圖，使用純色填充，
不要漸層，大小 200x200px，適合用作 Logo。
請直接輸出 SVG 代碼，不要解釋。
```

### 匯入 Vectorpea 的步驟

[flow]
1. 複製 AI 生成的 SVG 代碼
2. 開啟文字編輯器（記事本/TextEdit），貼上代碼
3. 存成 logo.svg（確認副檔名是 .svg，不是 .txt）
4. 在 Vectorpea 中 File → Open，選取 logo.svg
5. 所有路徑會以可編輯的向量物件匯入
[/flow]

> **驗證 SVG 是否正確**
> 匯入後如果看到正確的圖形，代表 SVG 代碼有效。如果什麼都沒有或出現錯誤，可能是 AI 生成了不完整的 SVG 語法，嘗試重新要求 AI 生成，或請它修正錯誤。

## 方法二：AI 向量化工具將圖片轉 SVG

### 點陣圖自動描圖（Auto-Trace）

這是最常見的「AI 圖片轉向量」工作流程：把 Midjourney、Stable Diffusion 等 AI 生成的 JPG/PNG，透過向量化工具轉換成 SVG。

**推薦工具**：
- **Recraft.ai**：AI 驅動，路徑較乾淨，適合複雜插圖
- **Adobe Express 向量化**：Adobe 生態系，與 Illustrator 相容性好
- **SVGConverteer.app**：免費，適合簡單圖形和 Logo

### 適合向量化的 AI 圖片特徵

[tags]
- [green] 適合：高對比度、邊緣清晰、顏色數量少、插圖風格
- [green] 適合：Logo、圖示、簡單人物剪影、幾何圖案
- [orange] 不適合：寫實照片、細碎紋理、漸層豐富的圖片
- [orange] 不適合：解析度過低（建議至少 512x512 像素以上）
[/tags]

### 向量化後匯入 Vectorpea

[flow]
1. 準備好 AI 生成的 JPG 或 PNG（建議用白色或透明背景）
2. 上傳到 Recraft.ai 或其他向量化工具
3. 下載輸出的 SVG 檔
4. 在 Vectorpea 中 File → Open 開啟 SVG
5. 開始在 Vectorpea 中整理路徑（見下一章節）
[/flow]

## 方法三：手動描圖（Manual Trace）

### 在 AI 圖片上層描繪向量路徑

當自動向量化效果不理想時，可以把 AI 圖片作為底圖，在上方手動用鋼筆工具描繪路徑。這個方法最費時，但結果最乾淨，適合 Logo 設計。

[callout type="tip" title="鋼筆工具實用技巧"]
- 描繪曲線時**節點越少越平滑**——盡量在曲線轉折處才放錨點
- 拖曳控制把手時按住 Shift 可限制角度
- 完成一段路徑後按 Enter 結束，避免多餘節點
[/callout]

[flow]
1. File → Open 開啟 AI 生成的圖片（JPG/PNG）
2. 在圖層面板中鎖定底圖圖層（避免誤選）
3. 新增一個圖層，在上方操作
4. 用鋼筆工具（P）沿著圖片輪廓描繪路徑
5. 完成後隱藏或刪除底圖圖層
[/flow]

# 路徑：節點編輯與路徑整理
> 向量化後的路徑通常很雜亂——整理節點是讓圖形印刷乾淨的關鍵步驟

## 節點編輯工具的基本操作

### 進入節點編輯模式

選取一個向量物件後，按 N 鍵切換到節點編輯工具（Node Edit Tool），或直接雙擊物件。此時可以看到路徑上所有的錨點，以及被選取錨點的控制把手。

**常見節點操作**：

| 操作 | 方法 |
|------|------|
| 選取單一節點 | 點擊錨點 |
| 選取多個節點 | Shift + 點擊，或框選 |
| 移動節點 | 拖移選取的錨點 |
| 刪除節點 | 選取後按 Delete（路徑會自動重新連接） |
| 新增節點 | 在路徑段上點擊 |
| 切換尖角/平滑 | 雙擊錨點 |

## 整理自動向量化的雜亂路徑

### 自動描圖的常見問題

自動向量化工具通常會產生過多節點，讓路徑看起來不平滑，縮放後邊緣呈鋸齒狀。整理的目標是：**用最少的節點描述最平滑的路徑**。

[flow]
1. 選取向量化後的物件，進入節點編輯模式（N）
2. 找出不必要的節點（直線段中間的節點、方向相同的連續節點）
3. 逐一刪除多餘節點，觀察路徑形狀是否維持
4. 調整剩餘節點的控制把手，讓曲線更平滑
5. 反覆檢查，直到路徑乾淨且自然
[/flow]

### 檢查開放路徑

印刷時封閉路徑才能正確填色。檢查路徑是否封閉：Object → Path → Close Path（如果路徑已封閉，選項為灰色）。向量化後的圖形有時會有意外斷開的路徑，需要手動連接。

## 路徑的布林運算

### 合併、相減、交集

多個形狀可以透過布林運算合併成新的形狀，在 Object → Path Operations 選單中找到：

**Unite（聯集）**：將多個形狀合併成一個，適合把分離的相同色塊合併。

**Subtract（差集）**：用上方的形狀裁切下方的形狀，適合製作鏤空效果（例如字母的負空間）。

**Intersect（交集）**：只保留兩個形狀重疊的區域。

> **印刷用途的布林運算**
> 送印前建議用 Unite 合併相同顏色的色塊，減少路徑數量，也避免印刷時出現色塊之間的白縫（叫做「漏白」，trapping 問題的一種）。

[reveal title="常見問題：為什麼兩個相鄰色塊之間會出現白縫？"]
這叫做「漏白」（trapping 問題）。原因通常是：
- 兩個色塊各自有微小的間隙（肉眼在螢幕上看不出來）
- 印刷裁切時的微小偏移讓間隙顯露

**解法**：使用 Unite（聯集）將相同顏色的相鄰色塊合併為單一路徑，徹底消除縫隙。
[/reveal]

# 排版：文字、形狀與版面設計
> Vectorpea 的排版能力讓你在同一個檔案裡完成設計和文字，無需再切換其他軟體

## 文字工具

### 建立文字的兩種方式

**點擊文字（Point Text）**：按 T 鍵選取文字工具，在畫布上點擊後輸入，文字不會自動換行，適合單行標題。

**區域文字（Area Text）**：按 T 鍵後在畫布上拖曳出一個矩形框，在框內輸入文字，超出框的文字會自動隱藏，適合段落正文。

### 字型設定

右側面板的 Character 區域控制：
- **Font Family（字體）**：選擇已安裝的字型，中文設計建議使用系統內建的繁體中文字型
- **Font Size（字級）**：以 pt 為單位，印刷標準的正文約 9–12pt，標題依版面而定
- **Line Height（行距）**：建議設定字級的 1.4–1.6 倍，例如 12pt 字配 17–20pt 行距
- **Letter Spacing（字距）**：標題可適度增加字距，正文保持預設即可

## 文字轉外框（Outline）

### 送印前必做的步驟

文字轉外框是指將文字圖層轉換為向量路徑，這樣印刷廠不需要安裝相同字型，就能正確印出你的字體設計。

[flow]
1. 確認文字內容和排版已定稿（轉外框後不能再編輯文字）
2. 選取所有文字物件（Ctrl + A 全選，或個別選取）
3. Text → Create Outlines（或 Shift + Ctrl + O）
4. 文字變為向量路徑，字形外框出現在路徑上
5. 送印前確認所有文字都已轉外框
[/flow]

> **轉外框前先存一份帶文字的備份**
> 轉外框後無法修改文字內容。建議先 File → Save 存一份帶文字的 SVG 工作檔，再轉外框後存另一份送印版本，命名加上「_outline」以區分。

[callout type="warning" title="轉外框是不可逆操作"]
文字一旦轉為外框，就變成普通向量路徑——**無法再修改內容、字型、字距**。
務必確認文字內容完全正確後再執行此步驟。
[/callout]

## 形狀工具與版面元素

### 基本形狀建立

左側工具列的形狀工具（按 Shift 切換）：
- **矩形工具（R）**：繪製矩形，按 Shift 繪製正方形
- **橢圓工具（E）**：繪製橢圓，按 Shift 繪製正圓
- **多邊形工具**：可設定邊數，繪製三角形、五角形等

所有形狀建立後可以在節點編輯模式下轉換為可編輯的路徑（Object → Convert to Path）。

### 對齊與分佈

選取多個物件後，右側面板的 Align 區域提供精準對齊工具：

[flow]
1. 選取要對齊的物件（Shift + 點擊多選）
2. 選擇對齊基準：對齊到選取範圍、對齊到畫板、對齊到特定物件
3. 點擊對應的對齊按鈕（左對齊、置中、右對齊、上下對齊）
4. 均勻分佈：水平分佈或垂直分佈，讓物件間距相等
[/flow]

## 版面設計原則

### 印刷排版的基礎規範

**出血（Bleed）**：設計尺寸四邊各加 3mm，確保裁切誤差不會露出白邊。在 File → Document Properties 設定畫板尺寸時記得加上出血。

**安全區域**：重要內容（文字、Logo）距離設計邊緣至少留 3–5mm，避免被裁切機切到。

**視覺層次**：
- 標題字級 > 副標題 > 正文
- 顏色對比明顯（深背景配淺文字，或反之）
- 留白（White Space）讓版面呼吸，不要塞滿

# 色彩：填色、漸層與印刷色彩
> 向量圖的色彩管理比點陣圖更直接，但印刷色彩設定有幾個關鍵點不能忽略

## 純色填色

### 設定填色的方式

選取物件後，在右側面板點擊 Fill 色塊，開啟色彩選擇器：

**色彩模式切換**：色彩選擇器上方可切換 RGB、HSL、十六進位（Hex）。如果印刷廠要求 CMYK 值，需要先在外部換算後輸入十六進位或 RGB 等效值（Vectorpea 目前介面以 RGB 為主）。

**透明度（Opacity）**：物件整體透明度在右側面板的 Opacity 滑桿控制，或直接在 Fill 色彩選擇器中調整 Alpha 值。

## 漸層填色

### 線性漸層與放射漸層

在填色選擇器中切換到 Gradient 模式：

**線性漸層（Linear）**：從一個方向到另一個方向的顏色過渡，可拖移漸層滑桿上的色標來調整顏色和位置。

**放射漸層（Radial）**：從中心點向外輻射的顏色過渡，適合製作光暈、立體球形感。

> **印刷漸層的注意事項**
> 印刷機的半色調網點對細微漸層不友善。漸層從深色到淺色時，接近白色的端點容易出現條帶感（Banding）。送印前建議在 PNG 輸出的校稿版本上仔細確認漸層效果。

## 筆畫（Stroke）設定

### 印刷用筆畫的重要設定

選取物件後，在右側面板的 Stroke 區域設定：

**Width（寬度）**：以 pt 或 px 設定。印刷品最細的筆畫建議不低於 0.25pt（約 0.088mm），過細的線條在印刷後可能斷線或完全消失。

**Cap（端點）**：
- Butt：端點切平，適合精確對齊
- Round：端點圓弧，適合手寫風格
- Square：端點超出路徑，讓視覺上線段稍長

**Join（轉角）**：
- Miter：尖角（適合幾何圖形）
- Round：圓角
- Bevel：斜角（避免尖角過長突出）

## 印刷色彩要點

### RGB 與印刷輸出

Vectorpea 的色彩介面以 RGB 為主，但大多數印刷廠接受 RGB 的 PDF（他們自行轉換），或要求 CMYK。送印前確認印刷廠的格式要求。

[tags]
- [blue] 數位輸出（網頁、螢幕）：使用 RGB
- [orange] 印刷輸出：詢問廠商要 RGB 還是 CMYK PDF
- [green] 安全做法：避免過度飽和的霓虹色，這類色彩印刷後會明顯偏暗
[/tags]

# 規格：工作畫板與印刷尺寸設定
> 在 Vectorpea 中從一開始就設定正確的畫板尺寸，遠比做完後再調整省力

## 設定工作畫板尺寸

### 開新檔案的正確流程

File → New Document，在對話框中設定：

**Width 與 Height**：輸入實際印刷成品尺寸，加上每邊 3mm 出血。例如 A4 傳單（21×29.7 公分）加出血後應輸入 21.6×30.3 公分。

**單位（Units）**：選 mm（公釐）或 cm（公分），印刷設計建議不要用 px。

**Color Mode**：選 RGB（Vectorpea 目前以 RGB 工作，匯出 PDF 時再確認色彩模式）。

### 常見印刷品尺寸（含出血）

| 印刷品 | 完成尺寸（mm） | 含出血尺寸（mm） |
|--------|--------------|----------------|
| 名片 | 90 × 54 | 96 × 60 |
| A5 傳單 | 148 × 210 | 154 × 216 |
| A4 傳單 | 210 × 297 | 216 × 303 |
| A3 海報 | 297 × 420 | 303 × 426 |

## 參考線設定

### 建立出血與安全區域的參考線

[flow]
1. 在頂部尺規上點擊並拖曳出水平或垂直參考線
2. 雙擊參考線可以輸入精確位置數值
3. 出血線：距離畫板邊緣 3mm 的位置（往外）
4. 安全線：距離畫板邊緣 3–5mm 的位置（往內）
5. View → Show Guides 切換參考線顯示
[/flow]

**鎖定參考線**：View → Lock Guides，避免操作時意外移動參考線。

## 多個畫板（Artboards）

### 一個檔案管理多個設計版本

Vectorpea 支援多個畫板，適合同一個印刷專案有多個尺寸（例如 A4 傳單 + 名片 + 社群貼文）。在 File → Document Properties 中新增畫板，或用工具列的畫板工具（A）直接在畫布上建立新畫板。

# 匯出：SVG、PDF 與印刷輸出
> 正確的匯出設定決定印刷品質——不同用途選不同格式

## 匯出前的送印檢查清單

[steps-status]
- [done] 所有文字已轉外框（Text → Create Outlines）
- [done] 筆畫寬度不小於 0.25pt
- [done] 畫板尺寸含出血（每邊 +3mm）
- [done] 重要元素距邊緣 3mm 以上（安全區域）
- [done] 向量路徑全部封閉（無意外開放路徑）
- [doing] 已存一份帶文字的 SVG 備份工作檔
- [todo] 漸層效果已在 PNG 校稿版本確認
[/steps-status]

## 匯出 PDF（送印首選）

### PDF 是向量印刷的標準格式

File → Export As → PDF，PDF 格式完整保留向量路徑、嵌入字型（如未轉外框）、色彩資訊，是印刷廠最通用的收稿格式。

[flow]
1. 確認文字已轉外框（或選擇嵌入字型）
2. File → Export As，選擇 PDF
3. 確認「Include bleed」已勾選（含出血匯出）
4. 若印刷廠要求 PDF/X-1a 或 PDF/X-4 規格，向他們確認具體設定
5. 存檔，命名加上「_print」後綴以區分
[/flow]

## 匯出 SVG（網頁與數位使用）

### SVG 保留完整向量可編輯性

File → Export As → SVG，SVG 格式在網頁上可以任意縮放，也可以用 CSS 或 JavaScript 動態控制樣式。適合網站 Logo、數位圖示等用途。

**SVG 匯出設定**：
- 建議勾選「Optimize」去除不必要的元數據，減少檔案大小
- 如果 SVG 要嵌入 HTML，確認移除 XML 宣告標頭（第一行的 `<?xml...>`）

## 匯出 PNG（校稿與社群用）

### 點陣化輸出供預覽確認

File → Export As → PNG，適合製作校稿版本給客戶確認，或用於社群媒體預覽。

**解析度設定**：
- 校稿用：150dpi 即可，檔案小方便傳送
- 社群媒體用：72–96dpi，1200×630px（Facebook/LinkedIn 推薦尺寸）
- 高品質備份：300dpi，保留最佳清晰度

## 送印工作流程總覽

[flow]
1. 確認需求：印刷廠規格（格式、色彩模式、出血、檔案大小上限）
2. 設定畫板：按完成尺寸 + 出血建立新文件
3. 置入設計元素：匯入 AI 生成的 SVG，或建立新向量圖形
4. 排版與文字：完成版面設計，正文和標題對齊、層次清晰
5. 轉外框：所有文字 Text → Create Outlines
6. 路徑整理：確認無開放路徑，合併相同色塊
7. 校稿：匯出 PNG 給客戶或自行確認
8. 送印：匯出 PDF（含出血），交給印刷廠
[/flow]

## 常見問題 FAQ

[accordion]
[item title="AI 生成的 SVG 可以直接送印嗎？" open]
不一定。AI 生成的 SVG 需要確認以下幾點：
- 路徑是否全部封閉（開放路徑無法正確填色）
- 是否有過多不必要的節點（影響印刷平滑度）
- 筆畫寬度是否符合印刷廠要求（不低於 0.25pt）
- 文字是否已轉外框（如果 SVG 包含文字）
建議匯入 Vectorpea 檢查並整理後再送印。
[/item]
[item title="RGB 和 CMYK 該用哪個？"]
Vectorpea 目前以 RGB 為主要工作模式。多數數位印刷廠接受 RGB 的 PDF，他們會自行轉換。但部分傳統印刷廠可能要求 CMYK 格式。
**安全做法**：送印前先確認印刷廠的規格要求，避免色彩偏差。
[/item]
[item title="為什麼印刷需要加出血？"]
印刷完成後需要裁切成成品尺寸，裁切機有微小的誤差。出血（每邊 3mm）確保裁切偏離時不會露出白邊。
沒有出血的設計，裁切後邊緣可能出現細白線，影響成品品質。
[/item]
[item title="Vectorpea 能開啟 Adobe Illustrator 的 .ai 檔嗎？"]
可以。Vectorpea 支援開啟 .ai（Adobe Illustrator）格式，以及 SVG 和 PDF。開啟後所有路徑會以可編輯的向量物件匯入。
[/item]
[item title="漸層印刷時出現條帶感（Banding）怎麼辦？"]
印刷機的半色調網點對細微漸層不友善，特別是接近白色的端點容易出現條帶。
**建議解法**：
- 在 PNG 校稿版本上仔細確認漸層效果
- 避免從深色直接過渡到純白的漸層
- 考慮改用純色或減少漸層跨度
[/item]
[/accordion]
