# 入門：認識 Photopea 工作環境
> Photopea 是免費的瀏覽器版影像編輯器，功能接近 Photoshop——不需要安裝，開啟網頁就能用

## 什麼是 Photopea？

### Photopea vs Photoshop：印前處理怎麼選

[compare label-left="Photoshop" label-right="Photopea"]

[/callout]

### 一個瀏覽器裡的 Photoshop

Photopea 是一款完全在瀏覽器中運行的專業影像編輯器，由單一開發者 Ivan Kutskir 自 2013 年持續開發。它的介面和操作邏輯幾乎與 Adobe Photoshop 相同，並且完整支援 PSD、XCF、Sketch 等專業格式。

[tags]
- [green] 完全免費，無需安裝
- [green] 支援 PSD 格式讀寫
- [blue] 介面與 Photoshop 高度相似
- [orange] 需要穩定的網路連線
[/tags]

### 印刷前處理為什麼選 Photopea？

AI 圖片生成工具（Midjourney、Stable Diffusion、DALL-E）輸出的圖片幾乎都是 RGB 模式、72dpi，直接送印會出現顏色偏差與解析度不足的問題。Photopea 提供了完整的工具鏈來解決這些問題，而且完全免費。

> **核心原則：AI 生圖只是起點，後處理決定最終品質**
> 任何 AI 工具生成的圖片都必須經過檢查、修復和規格調整，才能送交印刷廠。跳過這個步驟，印出來的結果往往令人失望。

## 工作介面導覽

### 四大區域認識

[flow]
1. 頂部選單列 — File、Edit、Image、Layer、Select、Filter 等功能選單
2. 左側工具列 — 選取、裁切、修復、筆刷、文字等工具（快速鍵與 Photoshop 相同）

### 最常用的面板

**圖層面板（Layers）**：顯示所有圖層的堆疊順序，是整個後處理流程的核心控制中心。如果看不到圖層面板，到 Window → Layers 開啟。

**屬性面板（Properties）**：選取調整圖層後，在此修改參數，例如曲線、色相飽和度的控制滑桿都在這裡。

**選項列（Options Bar）**：選取工具後，上方會顯示該工具的細部設定。例如選取工具的羽化程度、筆刷工具的不透明度，都在這裡調整。

### 三種開啟方式

# 基礎：圖層觀念與無損編輯
> 圖層是影像編輯的核心概念——理解它，你才能放心操作，不怕「改壞了回不去」

## 圖層是什麼？

### 透明膠片的比喻

把圖層想像成一疊透明膠片，每一張膠片上有不同的內容。疊在一起從上往下看，就是最終的畫面。你可以單獨調整某一張膠片，不影響其他層。

[flow]
1. 最底層 — AI 生成的原始圖片（背景圖層）
2. 中間層 — 調整圖層（曲線、色相等）
3. 上方層 — 修復筆刷的操作結果
4. 最上層 — 文字或 Logo 等設計元素
[/flow]

### 為什麼要用圖層？

直接在原始圖片上塗改，一旦存檔就無法還原。使用圖層結構，可以隨時關閉、刪除或修改某一層，而不影響原始圖片。這就是「非破壞性編輯」的核心原則。

> **第一條守則：永遠不直接編輯原始圖層**
> 開始任何操作之前，先按 Ctrl + J 複製圖層，在複本上操作。這樣無論發生什麼，原始圖片都還在。

## 三種關鍵圖層類型

### 像素圖層（一般圖層）

最常見的類型，儲存的是實際的像素資料。AI 生成的圖片開啟後就是像素圖層。直接在上面繪畫或修復，操作是永久性的（除非在歷史記錄範圍內還原）。

### 調整圖層（Adjustment Layer）

不儲存像素，只儲存調整參數，對它下方的所有圖層產生效果。在 Layer → New Adjustment Layer 選單中建立，或點選圖層面板底部的半圓圖示。

[tags]
- [green] 隨時修改參數，不破壞原圖
- [green] 可以用遮罩限制調整範圍
- [blue] 曲線、色相飽和度、色彩平衡都建議用調整圖層
- [orange] 不能直接在上面用筆刷繪畫
[/tags]

### 智慧物件（Smart Object）

將圖層封裝為智慧物件後，對它進行縮放、旋轉時不會損失畫質，因為 Photopea 記住了原始資料。方法：右鍵圖層 → Convert to Smart Object。

AI 生成的圖片如果需要縮放，建議先轉換為智慧物件再操作。

## 圖層操作基本功

### 必學的六個操作

[flow]
1. 複製圖層 — Ctrl + J（快速複製選取的圖層）
2. 新增空白圖層 — Ctrl + Shift + N（用於修復筆刷的操作）
3. 合併圖層 — Ctrl + E（合併選取的圖層，或 Ctrl + Shift + E 合併全部可見圖層）
4. 圖層不透明度 — 圖層面板右上角的 Opacity 數值（0% 完全透明，100% 完全不透明）
5. 圖層混合模式 — 不透明度旁的下拉選單（Normal、Multiply、Screen 等）
6. 圖層遮罩 — 圖層面板底部的方框圖示（後面章節詳述）
[/flow]

# 診斷：AI 圖片的常見印刷問題
> 收到 AI 生成的圖片後，先做「健康檢查」——知道問題在哪，才能對症下藥

[callout type="warning" title="印前三大致命錯誤"]
- 解析度不足：72dpi 直接送印，成品模糊馬賽克，客戶一眼就看出問題
- 色彩模式錯誤：RGB 圖片直接交稿，印出來顏色暗沉褪色，與螢幕差異巨大
- 忘記出血：設計貼邊卻沒加出血，裁切時露出白邊，整批報廢
[/callout]

## 解析度：印刷的第一道關卡

### 72dpi vs 300dpi 的差異

dpi（dots per inch）是每英寸的像素點數。螢幕顯示只需要 72–96dpi，但印刷需要至少 300dpi，否則放大後會看到明顯的馬賽克感。

**實際計算**：一張 1024×1024 像素的 AI 圖片，72dpi 下印刷尺寸約 36×36 公分，看起來沒問題；但實際印刷廠要求 300dpi，這張圖只能印到約 8.7×8.7 公分，超過就模糊。

> **檢查方式：Image → Image Size**
> 在 Image Size 對話框中，查看「Resolution」欄位。如果是 72 或 96，需要評估是否能夠放大（詳見章節 7）。

### 解析度、像素、印刷尺寸的關係

| AI 圖片像素 | 300dpi 可印最大尺寸（公分） | 適合用途 |
|-------------|--------------------------|---------|
| 1024 × 1024 | 8.7 × 8.7 | 名片、小貼紙 |
| 2048 × 2048 | 17.4 × 17.4 | A5 以下 |
| 4096 × 4096 | 34.7 × 34.7 | A3 左右 |

## 色彩模式：RGB 與 CMYK 的根本差異

### RGB vs CMYK 色彩模式對比

[compare label-left="RGB（螢幕用）" label-right="CMYK（印刷用）"]
- 光的三原色混合，色域廣 | 四色油墨疊印，色域較窄
- 可呈現霓虹色、螢光色 | 無法呈現極鮮豔的螢光色
- AI 圖片預設都是此模式 | 印刷廠通常要求的格式
- 直接送印會顏色偏差 | 轉換後色彩更貼近印刷成品
- 適用：螢幕展示、社群、網站 | 適用：傳單、名片、海報
[/compare]

### 為什麼顏色會「印出來不一樣」？

螢幕用光的三原色（紅 Red、綠 Green、藍 Blue）混合發光，理論上可以顯示任何顏色，包含非常鮮豔的霓虹色。印刷用四種油墨（青 Cyan、洋紅 Magenta、黃 Yellow、黑 Key）疊印，能呈現的色域比螢幕小。

當一張 RGB 的 AI 圖片直接轉成 CMYK 印刷，那些超出 CMYK 色域的鮮豔顏色會自動被壓暗、變灰，印出來看起來像「褪色」。

[tags]
- [blue] RGB：螢幕顯示，色域廣，支援霓虹色
- [orange] CMYK：印刷輸出，色域較窄，無法呈現極鮮豔色彩
- [green] 解法：提前在 RGB 模式調整，轉換前確認色彩不過飽和
[/tags]

### 如何檢查色彩模式

Image → Mode 選單，目前模式會打勾標示。AI 圖片通常是 RGB Color（8 bit），印刷廠通常要求 CMYK 或接受 RGB（各廠規定不同，先詢問）。

## AI 圖片常見視覺缺陷

### 典型問題清單

**手指與肢體異常**：這是目前 AI 生圖最常見的問題，手指數量不對、關節方向錯誤、手掌結構扭曲。如果印刷品的主角是人物，這是第一個要檢查的地方。

**文字失真**：AI 生圖器通常無法生成正確的文字，生成的「文字」往往是看起來像字但實際上無意義的圖形。不能直接把 AI 生成的文字當成資訊使用，必須在 Photopea 中重新打字覆蓋。

**邊緣鋸齒與光暈**：主體與背景交界處常出現不自然的光暈或鋸齒，在白色背景印刷品上尤其明顯。

**背景結構不一致**：遠處建築、地板紋路、牆面磁磚等重複性結構，AI 常生成出比例或透視錯誤的結果。

[tags]
- [orange] 高風險：手指異常、文字失真
- [orange] 中風險：邊緣光暈、背景結構錯誤
- [blue] 低風險：色彩偏差、局部模糊
[/tags]

### 快速診斷流程

[flow]
1. 縮放到 100% 實際大小，全圖瀏覽一遍
2. 特別放大人物手部區域（Ctrl + + 放大，空白鍵平移）
3. 檢查圖中是否有「文字」需要重打
4. 觀察主體邊緣是否有光暈或鋸齒
5. 記錄需要修復的區域，開始處理
[/flow]

# 修復：瑕疵清除技巧
> 三種工具對應三種情況——選對工具，修復工作事半功倍

## 污點修復筆刷（Spot Healing Brush）

### 適用場景

小範圍瑕疵的快速修復：臉上的小斑點、圖片中的雜訊點、背景上的小污點。這個工具會自動分析周邊紋理，直接填補，不需要手動指定來源。

**工具位置**：左側工具列，J 鍵快速鍵（與 Healing Brush 共用同一位置，可按 Shift + J 切換）。

### 操作方法

[flow]
1. 按 Ctrl + J 複製圖層，在複本上操作
2. 選取污點修復筆刷（J）
3. 在選項列調整筆刷大小，比要修復的區域略大一點
4. 直接在瑕疵上點擊或塗抹
5. Photopea 自動分析周邊紋理填補
[/flow]

> **小技巧：在空白圖層上操作**
> 新增一個空白圖層（Ctrl + Shift + N），在選項列勾選「Sample All Layers」，然後在空白圖層上操作。這樣修復結果在獨立圖層上，不滿意可以直接刪除，完全不影響原圖。

## 仿製印章（Clone Stamp）

### 適用場景

需要精確複製某個區域來覆蓋問題的情況：修復 AI 手指、移除背景中的多餘元素、延伸背景紋理。需要手動指定「來源點」，控制較精細。

**工具位置**：左側工具列，S 鍵快速鍵。

### 操作方法

[flow]
1. 選取仿製印章工具（S）
2. 調整筆刷大小與不透明度
3. 按住 Alt，點選要「複製」的來源區域
4. 放開 Alt，移到要覆蓋的問題區域塗抹
5. 來源點會跟著游標同步移動（注意對齊感）
[/flow]

### 修復 AI 手指的策略

AI 生圖的手指問題通常是多了一根或少了一根，或關節方向錯誤。處理策略：
- 放大到 200–400% 仔細觀察手指結構
- 用仿製印章從正確的手指部位複製，覆蓋多餘的手指
- 從旁邊的皮膚或背景複製來填補空白區域
- 邊緣用不透明度較低的筆刷做羽化過渡

[reveal title="仿製印章為什麼要按 Alt？"]
Alt 鍵是用來「取樣」來源點。按下 Alt 時游標會變成靶心圖示，此時點擊的區域就是你要複製的乾淨紋理。放開 Alt 後再塗抹到問題區域上，Photopea 就會把取樣的紋理搬過來。
- 取樣點要選擇與問題區域光線、角度、紋理都接近的地方
- 修復過程中要經常重新取樣（再按 Alt），避免紋理重複感太重
- 不透明度建議設在 70–90%，不要 100%，讓過渡更自然
[/reveal]

## 修補工具（Patch Tool）

### 適用場景

大面積區域的置換：移除圖中的人物、替換大塊背景、處理AI生成的錯誤建築結構。先框選問題區域，再拖移到乾淨的來源區域。

**工具位置**：左側工具列，按 Shift + J 在 Healing 系列工具中切換找到 Patch Tool。

### 操作方法

[flow]
1. 選取修補工具
2. 在選項列確認模式為「Normal」，來源為「Source」
3. 用工具圈選要移除的問題區域
4. 將選取區域拖移到旁邊紋理相似的乾淨區域
5. 放開滑鼠，Photopea 自動混合邊緣
6. Ctrl + D 取消選取查看結果
[/flow]

## AI 文字的處理方式

### 為什麼 AI 生成的文字不能用

AI 圖片中看起來像文字的部分，實際上是圖形，沒有語意。如果印刷品需要顯示正確資訊，必須：

[flow]
1. 用仿製印章或修補工具，將 AI 生成的「文字圖形」覆蓋掉
2. 用文字工具（T）重新輸入正確文字
3. 調整字型、大小、顏色，確保與整體設計協調
[/flow]

# 選取：精準去背與遮罩
> 精準的選取是一切後期合成的基礎——選得準，才能改得漂亮

## 選取工具的選擇邏輯

### 依照主體與背景的對比度選擇

[flow]
1. 主體與背景顏色反差大 → 魔術棒或顏色範圍
2. 主體輪廓清晰但不規則 → 快速選取工具
3. 主體有細碎邊緣（頭髮、毛皮）→ Select and Mask 的細化邊緣
4. 需要手動精確控制 → 鋼筆工具（路徑）
[/flow]

## 快速選取工具（Quick Selection）

### 操作方法

工具快速鍵 W，在主體上塗抹，Photopea 會自動偵測邊緣並擴展選取範圍。如果選多了，按住 Alt 在多選的區域塗抹來扣除。

選取完成後，按 Select → Select and Mask 進入邊緣細化介面，處理鋸齒和光暈問題。

## 魔術棒與顏色範圍

### 魔術棒（Magic Wand）

工具快速鍵 W（按 Shift + W 切換），點擊某個顏色區域，自動選取相近顏色的範圍。Tolerance（容差）值越大，選取的顏色範圍越廣。

適合去除純色背景（如 AI 生成的白底或純色背景圖），一鍵點選背景後反選（Ctrl + Shift + I）即可選到主體。

### 顏色範圍（Color Range）

Select → Color Range，可以用滴管點選想要的顏色，Fuzziness（模糊度）控制相似程度。比魔術棒更精準，可以預覽選取結果。

## 選取並遮住（Select and Mask）

### 邊緣細化的關鍵設定

進入 Select → Select and Mask 後，右側有幾個重要設定：

**Radius（半徑）**：Photopea 在邊緣外側多少像素範圍內搜尋細節。頭髮邊緣設定大一點（10–15px），硬邊物件設定小一點（1–3px）。

**Smart Radius（智慧半徑）**：勾選後自動判斷邊緣類型，軟硬邊都能處理，建議預設開啟。

**Feather（羽化）**：讓邊緣稍微模糊，使合成看起來更自然，通常設定 0.5–1px。

**Decontaminate Colors（去色污染）**：移除邊緣殘留的背景顏色，去白底時特別有用。

[flow]
1. 用快速選取工具大略選出主體
2. 進入 Select and Mask
3. 開啟 Smart Radius，調整 Radius 值
4. 用邊緣細化筆刷（左側第二個筆刷工具）塗抹頭髮等細碎邊緣
5. 設定 Output 為「New Layer with Layer Mask」輸出
[/flow]

## 圖層遮罩 vs 橡皮擦

### 為什麼不要用橡皮擦去背

橡皮擦直接刪除像素，操作後無法還原（除非在歷史記錄範圍內）。圖層遮罩是用黑白畫面控制圖層的顯示範圍，黑色遮蓋、白色顯示，隨時可以修改。

> **守則：去背一律使用圖層遮罩，不用橡皮擦**
> 遮罩可以隨時修改或刪除，橡皮擦的操作是永久的。這個原則在印刷前處理尤其重要，因為客戶經常在最後要求微調邊緣。

[quiz type="bool"]
Q: 去背時使用橡皮擦比圖層遮罩更方便，因為可以直接看到結果？
- [ ] 是
- [x] 否
Hint: 思考「非破壞性編輯」的原則
[/quiz]

[reveal title="橡皮擦 vs 遮罩，实际操作差多少？"]
假設你去背後發現邊緣切得太緊，頭髮少了一根：
- 用橡皮擦：該區域像素已刪除，只能靠 Ctrl + Z 還原（如果超過 20 步就回不去了），或者從頭重做
- 用遮罩：只要用白色筆刷在遮罩上塗抹，被遮蓋的頭髮就回來了，30 秒搞定
遮罩的本質是一張黑白圖片，黑遮白顯，改黑白就等於改顯示範圍，完全不動到原始像素。
[/reveal]

# 調色：讓 AI 圖片符合印刷要求
> AI 圖片的色彩通常針對螢幕優化，偏鮮豔；印刷需要稍微收斂，才能忠實還原

## 調整圖層的正確使用方式

### 永遠用調整圖層，不用 Image 選單

Image → Adjustments 選單下的調整是直接作用在像素上，不可逆。應該改用 Layer → New Adjustment Layer，或點選圖層面板底部的半圓圖示建立調整圖層。

調整圖層的效果可以隨時修改參數，也可以用遮罩限制影響範圍（例如只調整背景，不影響主體）。

## 亮度與對比

### 基礎調整入門

建立「亮度/對比」（Brightness/Contrast）調整圖層，適合快速修正整體曝光問題。AI 圖片有時會過亮或對比過強，影響印刷後的可讀性。

[tags]
- [green] 亮度（Brightness）：正值提亮，負值壓暗
- [blue] 對比（Contrast）：正值增強，負值降低
- [orange] 印刷建議：對比不要設定太高，陰影部分油墨量容易過重
[/tags]

## 曲線（Curves）

### 最精準的色調控制工具

建立「曲線」（Curves）調整圖層，比亮度/對比更精細。曲線圖的橫軸是輸入值（原始亮度），縱軸是輸出值（調整後亮度）。

**S 型曲線**：同時提亮高光、壓暗陰影，增加立體感。最常用的基本調整。

**針對印刷的調整策略**：
- 高光部分（曲線右上角）略微壓低，避免白色區域印刷時看不到細節
- 陰影部分（曲線左下角）略微提高，避免深色區域油墨疊印過重、紙張吃墨不均

### 分色頻道調整

曲線下拉選單可以切換 RGB 總控，或單獨調整 Red、Green、Blue 頻道，用來修正偏色問題。例如圖片偏黃，可以在 Blue 頻道的高光部位略為提高。

## 色相/飽和度（Hue/Saturation）

### 處理 AI 圖片的高飽和問題

Midjourney 等 AI 工具生成的圖片通常飽和度偏高，轉為 CMYK 後這些高飽和色彩會被壓縮，印出來反而看起來暗沉。建議在轉 CMYK 前先降低飽和度。

建立「色相/飽和度」調整圖層，將 Saturation 滑桿往左移動 -10 到 -20，讓色彩稍微收斂，印刷後更接近螢幕呈現的效果。

**針對特定顏色調整**：下拉選單由「Master」切換到「Reds」、「Blues」等，可以單獨調整某個顏色的飽和度，例如只降低過度鮮豔的天空藍。

## 色彩平衡（Color Balance）

### 修正整體色調偏移

建立「色彩平衡」調整圖層，可以分別對 Shadows（陰影）、Midtones（中間調）、Highlights（高光）做色調調整。

常見用途：AI 生圖的室內場景常偏橘黃（暖色），如果需要偏冷色調的風格，在中間調略微增加 Cyan 和 Blue。

# 規格：印刷標準與輸出設定
> 這是送印前最關鍵的一步——尺寸、解析度、色彩模式，三個數字決定印刷成敗

## 調整解析度至 300dpi

### Image Size 的正確操作

Image → Image Size 打開對話框，注意兩個設定：

**Resample（重新取樣）**：決定 Photopea 如何在像素不夠時「補像素」。印刷放大時選「Preserve Details 2.0」（效果最好），縮小時選「Bicubic Sharper」。

**不勾選 Resample 時**：只是改變「每英寸像素數」的標籤，圖片的實際像素數量不變。這等於是直接告訴印表機「請用 300dpi 解讀這張圖」，如果原圖像素不夠，印出來還是模糊。

### AI 圖片放大策略

如果 AI 圖片解析度不足，有幾個選擇：

[flow]
1. 評估所需的印刷尺寸，計算需要多少像素
2. 如果差距不大（例如需要 105%），可以在 Photopea 用 Preserve Details 放大
3. 如果差距較大（超過 120%），建議用 AI 放大工具（Topaz Gigapixel、Magnific AI）先放大再導入 Photopea
4. 或者縮小印刷尺寸，在原始解析度內完成
[/flow]

## 常見印刷尺寸規格

### 台灣常用印刷尺寸

| 印刷品 | 常用尺寸（公分）| 300dpi 所需像素 |
|--------|----------------|----------------|
| 名片 | 9 × 5.4 | 1063 × 638 |
| A4 傳單 | 21 × 29.7 | 2480 × 3508 |
| A3 海報 | 29.7 × 42 | 3508 × 4961 |
| 橫幅（90×180） | 90 × 180 | 需分割或降至 150dpi |

**大型輸出特例**：展覽看板、戶外廣告等觀看距離遠的大型輸出，通常允許 150dpi 甚至 72dpi，因為觀看距離拉開後，像素不足的問題不明顯。

## 出血設定（Bleed）

### 為什麼需要出血

印刷品裁切時有 1–2mm 的誤差，如果設計只到邊緣，裁切偏差會露出白邊。「出血」是在設計尺寸外圍多延伸的安全區域，通常是三邊各加 3mm。

### 在 Photopea 中設定出血

[flow]
1. 確認你的設計尺寸（例如 A4：21 × 29.7 公分）
2. Image → Canvas Size（注意不是 Image Size）
3. 將寬度加 0.6cm（左右各 3mm），高度加 0.6cm（上下各 3mm）
4. Anchor 設定為中心點
5. 確認，畫布擴大後在四周會看到延伸出來的區域
6. 確保背景圖片填滿整個擴大後的畫布
[/flow]

**安全區域**：重要內容（文字、Logo）距離設計邊緣至少留 3–5mm，避免裁切時被切到。

## RGB 轉 CMYK

### 轉換時機

轉換 CMYK 應在所有調色工作完成之後。轉換後色域會縮小，部分顏色會變化，所以順序是：先把 RGB 圖片調整到最理想狀態，最後再轉換。

### 轉換步驟

[flow]
1. 所有調整圖層的效果確認滿意
2. Image → Mode → CMYK Color
3. Photopea 會提示合併調整圖層，選「OK」
4. 觀察顏色變化，特別注意是否有明顯偏色
5. 如果某個顏色變化過大，可以在 CMYK 模式下再次用色彩平衡微調
[/flow]

> **注意：各印刷廠規定不同**
> 有些印刷廠接受 RGB 檔案（他們自行轉換），有些要求 CMYK。送印前務必確認廠商規格，按照廠商指定的設定處理。

### 色域警告

在 RGB 模式下，可以到 View → Gamut Warning 開啟色域警告，超出 CMYK 色域的顏色會顯示灰色遮罩，讓你提前知道哪些顏色轉換後會有明顯變化。

# 匯出：完成送印前的最終輸出
> 最後一關——選對格式，確認所有設定，讓檔案以最完整的狀態交到印刷廠

## 存檔 PSD（保留圖層的工作檔）

### 先存 PSD 再匯出

在匯出任何印刷格式之前，先 File → Save 存一份 PSD。PSD 保留所有圖層、遮罩、調整圖層，是可以繼續修改的工作檔。印刷廠退回要求修改時，可以直接打開 PSD 調整。

**檔案命名建議**：`專案名稱_YYYYMMDD_工作檔.psd`，日期讓你在多版本修改時容易辨識最新版。

## 匯出 PDF（印刷用）

### PDF 是印刷廠最通用的格式

File → Export As → PDF（或 File → Save as，選 PDF 格式），PDF 格式可以嵌入字型、保留色彩模式，是印刷業的標準交稿格式。

[flow]
1. File → Export As，下拉選單選 PDF
2. 確認「Embed fonts」已勾選（如果有文字圖層）
3. Quality 設定：印刷用選最高品質
4. 確認色彩模式是 CMYK（如果廠商要求）
5. 存檔，檔名加上「_print」後綴以區分用途
[/flow]

## 匯出 JPEG（校稿用）

### 提供給客戶確認的版本

File → Export As → JPEG，匯出供客戶在螢幕上確認的低解析度版本。

[tags]
- [green] Quality 85–90：品質與檔案大小的平衡點
- [blue] 解析度可降至 150dpi：螢幕確認用不需要 300dpi
- [orange] 不要用 JPEG 送印：每次存 JPEG 都會損失品質
[/tags]

## 匯出 PNG（含透明底）

### 需要透明背景時使用

如果去背後的圖片需要保留透明，只能用 PNG（JPEG 不支援透明）。File → Export As → PNG，確認背景圖層已隱藏或刪除，透明區域會顯示為棋盤格紋路。

## 送印前最終檢查

### 送印檢查清單

[steps-status]
- [done] 解析度確認 300dpi（Image → Image Size 查看）
- [done] 色彩模式確認（CMYK 或按廠商指定）
- [done] 畫布尺寸含出血（設計尺寸 + 每邊 3mm）
- [done] 重要文字與圖案距邊緣至少 3mm（安全區域）
- [done] 文字圖層確認已轉外框或 PDF 嵌入字型
- [done] AI 缺陷已修復（手指、文字、邊緣光暈）
- [doing] 存有 PSD 工作檔備份（File → Save）
- [todo] 匯出 PDF 給印刷廠，JPEG 給客戶校稿
[/steps-status]
