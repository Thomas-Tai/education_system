認識 Leonardo.ai

它是甚麼？

Leonardo.ai 是一個強大的 AI 圖像生成工具。你只需要用中文寫描述，它就能幫你畫出精美的圖像。不用擔心英文！

操作只要 3 步

  1. ❶ 寫咒語 — 在下方輸入框用中文寫下你想要的角色描述
  2. ❷ 翻譯 — 點 Improve Prompt 自動轉英文
  3. ❸ 生成 — 點 Generate 等幾秒就出圖！
帳號問題：帳號已由老師預先準備好。如果遇到登入問題，請舉手，助教會協助你。
❶ 輸入框在畫面底部

實戰篇25 - 35 分鐘設計你的原創角色

你的創作任務

你的任務：設計一位原創角色 先想一想： 1. 他 / 她的名字和背景故事？ 2. 有甚麼超能力或特殊技能？ 3. 個性是開朗還是沉穩？ 4. 裝備和外表長甚麼樣？
先和夥伴討論，再寫出最精準的 Prompt！
不知道從哪裡開始？選一個模板，填入你的創意：
模板 A（科幻）：「一位______的星際探險家，______（特徵），穿著______（裝備），站在______（場景），科幻動漫風格」
模板 B（奇幻）：「一位______的精靈戰士，______（特徵），手持______（武器），在______（場景）戰鬥，吉卜力風格」
模板 C（自由）：從零開始！用四要素公式自己寫出最強咒語

創作流程

  1. 創意總監發想角色概念
  2. 技術總監將概念轉化為 Prompt
  3. 用 Leonardo.ai 生成圖像
  4. 不斷 Debug，越改越好！
  5. 選出最滿意的最終版本
