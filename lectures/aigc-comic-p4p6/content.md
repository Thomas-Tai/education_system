在每一個頂尖的設計工作室裡，都有明確的團隊分工。今天的課程中，親子將組成一個雙人設計工作室，各自扮演關鍵角色：

[compare label-left="創意總監 (Creative Director)" label-right="技術總監 (Technical Director)"]
- 負責腦力風暴，發想最酷的故事與最帥的角色 | 負責魔法詠唱，將創意轉化為 AI 能懂的工程指令
- 決定角色的名字、背景故事、超能力與個性 | 將天馬行空的想法精準地寫成 Prompt 指令

# 咒語篇：掌握指令的藝術
> 學會 Prompt 工程的四要素公式，讓你的指令從普通升級為傳說級

## 四要素 Prompt 公式

專業人士把「下指令」這件事稱為 Prompt Engineering -- 指令工程學。其實它有公式可循，一個好的 Prompt 至少包含四個部分：

[image-text position="left" width="40"]
![Prompt 四要素公式](assets/images/2f1d3baa-8e90-4f22-be04-fd38df852618.png)
把四個要素像拼圖一樣組合在一起，就能產出令人驚豔的結果。缺少任何一個要素，AI 的創作就會變得模糊、缺乏方向。
[/image-text]

[flow]
1. 核心主體 (Subject) -- 你的主角是誰？例如：一位年輕的魔法師
2. 細節特徵 (Details) -- 他有什麼特徵？例如：戴著星星圖案的帽子，藍色眼睛
3. 環境背景 (Background) -- 他在哪裡？例如：站在古老的圖書館裡
4. 藝術風格 (Style) -- 你想要什麼風格？例如：日本動漫風格
[/flow]

把這四個元素組合起來，就像在調配一瓶魔法藥水。範例如下：

```prompt [label="完整 Prompt 範例"]
一位年輕的魔法師，戴著星星圖案的帽子，藍色眼睛，站在古老的圖書館裡，日本動漫風格
```

## 咒語升級挑戰 -- 從 LV.1 到 LV.99

建構指令就像是遊戲裡的角色升級。每一層新增的元素，都讓你的指令更強大：

[image-text position="right" width="45"]
![咒語升級公式](assets/images/9f5577af-338d-4588-9e46-e4b749e7da52.png)
- **LV.1 咒語**："一個男孩" -- 很普通
- **+ 細節特徵** -- 角色升級
- **+ 環境背景** -- 戰鬥力 UP
- **+ 藝術風格** -- 史詩級進化
- **= LV.99 傳說級咒語**
[/image-text]

### 升級前後對比

[compare label-left="LV.1 普通咒語" label-right="LV.99 傳說級咒語"]
- Prompt: "a boy" | Prompt: "a boy with silver hair, glowing blue eyes, wearing futuristic armor with neon patterns"
- 結果：很普通、沒有特色 | 結果：高度細節、動態感十足的角色設計
- 缺乏個性和故事感 | 充滿故事背景和視覺張力
[/compare]

Hint: LV.99 的咒語要同時包含主體、細節、背景和風格四個要素。
[/quiz]

## 風格庫 -- 為你的角色換上 SKIN

風格決定了你角色的靈魂。同樣的角色，用不同的風格指令，會產生完全不同的感覺。以下是六大超好用的風格關鍵字，把它們當作你的技能卡：

[tags]
- [blue] Anime -- 動漫風：熱血、大眼睛、鮮豔色彩
- [purple] Cyberpunk -- 賽博龐克：霓虹燈、高科技、暗色調
- [orange] Concept Art -- 遊戲概念藝術：精細、寫實、專業設定感
- [green] Watercolor -- 水彩風：柔和、渲染、藝術氣質
- [purple] Comic Book -- 美式漫畫：粗線條、高對比、動態張力
- [blue] Studio Ghibli -- 吉卜力風格：溫暖、奇幻、手繪質感
[/tags]

[image-text position="left" width="45"]
![六大風格選擇](assets/images/Gemini_Generated_Image_mxh20vmxh20vmxh2.png)
每一張風格卡都像一把能改變世界樣貌的武器。在你的咒語最後加上一張你最喜歡的風格卡，看看會發生什麼神奇的化學反應。

創意總監和技術總監請先討論，為你們等一下的專案挑選一到兩種最適合的風格裝備。
[/image-text]

## 實用工具 -- Leonardo.ai

Leonardo.ai 是今天我們要使用的 AI 繪圖平台。它有一個非常方便的功能：你可以先輸入中文描述，然後使用 Improve Prompt 功能，讓系統自動幫你轉換成專業的英文 Prompt。

![Leonardo.ai 介面](assets/images/Screenshot_2025-10-24_193216.png)

[callout type="info" title="操作小技巧"]
不需要擔心英文不夠好。先用中文把你想像的畫面完整描述出來，再按 Improve Prompt 按鈕，系統就會幫你轉成 AI 最容易理解的英文指令。
[/callout]

[bonus title="延伸閱讀：Prompt 進階技巧"]
- 使用 negative prompt 排除你不想要的元素，例如 "no text, no watermark"
- 加入 "highly detailed" 或 "4K" 可以提升畫面精細度
- 嘗試組合多種風格，例如 "anime style with cyberpunk elements"
- 用 "cinematic lighting" 或 "dramatic angle" 增加畫面的戲劇感
[/bonus]

- **六大風格裝備** | Anime、Cyberpunk、Concept Art、Watercolor、Comic Book、Ghibli

## 你的創作任務

![任務卷軸](assets/images/Gemini_Generated_Image_wxqbj8wxqbj8wxqb.png)

在下指令之前，先花點時間和夥伴討論以下問題。這些思考會幫助你的 Prompt 更有深度，讓 AI 創造出來的角色不只好看，更有靈魂：

### 角色設計思考清單

- **名字與背景**：你的角色叫什麼名字？來自哪個星球或文明？
- **超能力或技能**：他有什麼獨特的能力？是戰鬥型、智慧型還是探索型？
- **個性特質**：他是開朗外向還是沉穩內斂？勇敢中帶有神秘感要如何表現？

[/callout]

## Debug 時間 -- 迭代優化

在設計界，沒有一次就成功的。我們把「不斷修改、越改越好」的過程叫做「迭代」。AI 有時候會聽錯指令，畫出一些搞笑的東西，但這正是 Debug 的機會。

[image-text position="left" width="45"]
![迭代優化前後對比](assets/images/e31750f5-d776-47b9-b6d5-ad16961a5625.png)
**V1.0 (有 Bug)**：Prompt 只有 "a boy"，結果很普通。

**V2.0 (Bug 修復)**：加入 silver hair, glowing eyes, armor 等細節，角色立刻有了生命。

每一次的修改都是一次版本更新。有時候 AI 意外跑出來的「Bug」，反而會給你更有趣的靈感。
[/image-text]

### 迭代優化的三個方向

- **細節不夠？** 檢查是否缺少服裝、配件、表情、姿勢等描述
- **風格不對？** 確認藝術風格關鍵字是否明確，例如改用 Concept Art 或 Anime
- **背景太单调？** 加入場景描述，例如「站在星際太空站的觀景台，窗外是銀河」

Hint: 最好的做法是在現有 Prompt 基礎上進行 Debug 迭代，逐步修正細節，而不是全部重來。

# 交付篇：展示你的傑作
> 選出最佳作品、簽署交付、裝裱典藏，並在創作者聚光燈下分享你的設計故事

## 作品交付流程

![作品簽署與交付](assets/images/Gemini_Generated_Image_a5dicea5dicea5di.png)

當你對角色設計感到滿意時，就該進行正式交付。這不只是交作業，而是你作為設計師的第一號正式作品。

[steps-status]
- [done] 腦力激盪 | 與夥伴討論角色設定與 Prompt
- [done] 迭代優化 | 經過多次 Debug 達到最佳版本
- [doing] 選出最佳作品 | 從所有版本中挑選你最滿意的一張
- [todo] 簽署與裝裱 | 在作品角落簽上工作室大名，裝入畫框
[/steps-status]

### 創作典藏三步驟

- **Finalize**：敲定你的最終版本，確認這就是你要的設計
- **Print**：高解析度列印，確保畫面品質
- **Archive**：裝裱入框，成為你的第一份作品集

[callout type="tip" title="簽署儀式"]
請創意總監和技術總監一起，在你們作品的角落簽上工作室的大名。這代表了你們對這件作品的認可與驕傲。從今天起，這就是你們工作室的第一號正式作品。
[/callout]
